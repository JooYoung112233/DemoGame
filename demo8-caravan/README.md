# Demo 8: Caravan

## 컨셉
선형 스토리 캠페인 전투. 12개 노드로 구성된 캠페인 맵을 따라 캐러밴을 이끌고, 유닛을 모집/전직하며 마왕성까지 진군한다.

## 핵심 루프
**마을(모집/업그레이드) → 전투 → 이벤트(선택) → 다음 노드 → 반복 → 보스**

---

| # | 타입 | 내용 | 보상 |
|---|------|------|------|

## 유닛 전직 트리

### 근접 라인
```
병사 ─┬─ 기사 ── 성기사 (탱크 특화)
      └─ 광전사 ── 전쟁군주 (DPS 특화)
```
- 기사: Shield Block → 성기사: Holy Strike
- 광전사: Frenzy → 전쟁군주: War Cry

### 원거리 라인
```
궁수 ─┬─ 저격수 ── 명사수 (크리 특화)
      └─ 화염 궁수 ── 파이로맨서 (AoE 특화)
```
- 저격수: Precise Shot → 명사수: Pierce
- 화염 궁수: Fire Arrow → 파이로맨서: Firestorm

### 힐러 라인
```
성직자 ─┬─ 사제 ── 대사제 (순수 힐)
        └─ 전투 성직자 ── 이단심문관 (하이브리드)
```
- 사제: Heal → 대사제: Grand Heal
- 전투 성직자: Judgment

### 전직 비용
- 1차 전직: 30G
- 2차 전직: 50~80G

## 이벤트 시스템

| 이벤트 | 선택지 A | 선택지 B |
|--------|----------|----------|
| 떠돌이 상인 | -25G → 전체 회복 | 거절 |

## 자원 관리

- **골드**: 모집, 회복, 전직, 이벤트에 사용
- **모집**: 마을마다 다른 유닛 풀
- **전직**: 골드 소모로 상위 클래스 승급
- **전투 기록**: 유닛별 총 피해량, 총 회복량 추적
