# Demo 5: Blood Pit

## 컨셉
로그라이크 자동전투 파밍 던전. 로스터를 구성해 블러드 핏에 투입하고, 20라운드를 돌며 장비와 자원을 파밍한다. 전투는 자동이며, 전략은 파티 편성/장비 선택/탈출 타이밍에 집중.

## 핵심 루프
**로스터 편성 → 전투(자동) → 보상 선택 → 계속/탈출 판단 → 반복**

---

## 클래스 시스템

### 기본 클래스 (8종)
| 클래스 | 역할 | 특징 |
|--------|------|------|
| 전사 | 탱커 | 높은 HP/DEF, 느린 공속 |
| 성기사 | 탱커 | 밸런스형 탱커 |
| 도적 | 근접 DPS | 빠른 공속, 출혈 확률 |
| 암살자 | 근접 DPS | 최고 크리율/크리뎀, 회피 |
| 마법사 | 원거리 DPS | 높은 ATK, 넓은 사거리 |
| 궁수 | 원거리 DPS | 밸런스형 원거리 |
| 사제 | 힐러 | 기본 회복 |
| 주술사 | 힐러 | 보조형 힐러 |

### 전직 (Lv.5 + 골드)
각 기본 클래스 → 상위 클래스 1종
- 전사 → 광전사 (흡혈 추가)
- 도적 → 검성 (크리/출혈 강화)
- 사제 → 주교 (힐 강화)
- 마법사 → 대마법사 (극딜)
- 궁수 → 저격수 (초장거리 크리)
- 성기사 → 십자군 (방어 특화)
- 암살자 → 환영 (극한 회피/크리)
- 주술사 → 위치닥터 (출혈+힐 복합)

### 캐릭터 이름 생성
접두사(15종) + 접미사(15종) 조합으로 자동 생성
예: "피바람 바르크", "어둠의 세라"

## 드랍/아이템

### 무기 (5종)
- 주먹 (기본), 검, 도끼, 창, 샷건
- 희귀도에 따라 스탯 배율 적용 (1.0x ~ 4.0x)

### 패시브 (15종)
- **출혈 빌드**: 출혈 강화, 출혈 폭발, 출혈 추가뎀
- **회피 빌드**: 회피 강화, 회피 반격, 회피 공속
- **시체 빌드**: 시체 폭발, 시체 흡수
- **일반**: 공격력/공속/HP/크리율/크리뎀/흡혈/가시 증가

### 소비 아이템
- 붕대 (전체 HP 30% 회복)
- 폭탄 (적 전체 150 피해)
- 해독제 (상태이상 해제 + DEF+5)

### 희귀도 (7단계)
일반 → 고급 → 희귀 → 에픽 → 전설 → 고대 → 신화

## 런 구조 (20라운드)

RunManager가 라운드 구성을 생성:
- **전투**: 일반 전투 (위험도 스케일링)
- **보스**: 핏로드 전투
- **상점**: 아이템 구매/판매
- **대장간**: 장비 강화/조합
- **이벤트**: 랜덤 이벤트

### 파밍 시스템
- **레이드 인벤토리**: 전투 중 획득한 아이템
- **보안 컨테이너** (3칸): 사망해도 보존
- **탈출킷**: 분해 시 확률 획득, 사용 시 보상 보존 탈출
- **분해**: 아이템 → 골드 + 탈출킷 확률

### 위험도 스케일링
라운드가 진행될수록 적 스탯 상승, 동시에 드랍 희귀도도 상승

## 씬 구성
| 씬 | 역할 |
|----|------|
| SetupScene | 로스터 관리, 파티 편성, 징집, 대장간, 스태시 |
| BattleScene | 자동전투 실행 |
| DropScene | 보상 카드 선택 (3장 중 1장) |
| ChoiceScene | 계속 전투 vs 탈출 판단 |
| ShopScene | 아이템 구매/판매 |
| ForgeScene | 강화/조합 |
| EventScene | 랜덤 이벤트 |
| ResultScene | 런 결과 (승리/패배/탈출) |

## 기술 구현
- Phaser 3 Canvas, 번들러 없음
- 캐릭터는 Graphics API로 프로시저럴 렌더링 (역할별 실루엣 차별화)
- 아이콘도 Graphics API로 렌더링 (IconRenderer.js)
- 1280x720 해상도
