# Demo 7: Blackout

## 컨셉
정전된 건물 탈출 로그라이크. 5x5 그리드의 어두운 방을 탐험하며 출구를 찾아 탈출한다. 진행할수록 긴장도가 올라 적 출현 확률이 증가하고, 무게 관리와 탈출 타이밍이 핵심.

## 핵심 루프
**방 이동 → 방 타입에 따른 이벤트 처리 → 아이템/전투/함정 → 출구 도달**

---

## 맵 시스템

### 구조
- **5×5 그리드** (25개 방)
- **입구**: 좌측 (x=0, y=2)
- **출구**: 우측 (x=4, 랜덤 y), 재방문 시 위치 변경
- **잠긴 방**: 1개 (열쇠/락픽 필요)
- **전장의 안개**: 인접한 방만 표시

### 이동
- 4방향 이동 (상하좌우)
- 인접한 방으로만 이동 가능

---

## 방 타입 (8종)

| 타입 | 기본 확률 | 긴장도 영향 | 설명 |
|------|----------|------------|------|
| 빈 방 | 40% | 긴장도 높으면 감소 | 안전 |
| 적 방 | 25% | +긴장도×5% | 전투 발생 |
| 아이템 방 | 30% | -긴장도×2% | 장비/소비 획득 |
| 함정 방 | 10% | +긴장도×3% | 피해/디버프 |
| 입구 | 고정 | - | 시작 지점 |
| 출구 | 고정 | - | 탈출 목표 |
| 잠긴 방 | 1개 고정 | - | 열쇠 필요 |
| 안전 방 | 5% | - | 회복 가능 |

---

## 긴장도 시스템

탐험 깊이(x좌표)에 비례해 긴장도 상승:
- **적 출현 확률**: 25% + 긴장도 × 5%
- **아이템 확률**: 30% - 긴장도 × 2%
- **함정 확률**: 10% + 긴장도 × 3%
- **적 스탯 스케일링**: 1 + 깊이×0.15 + 긴장도×0.10

## 무게/인벤토리

- 캐릭터별 소지 용량 존재
- 방에서 발견한 아이템은 무게 차지
- 초과 시 이동/전투 효율 저하
- 버릴 아이템 vs 가져갈 아이템 전략적 선택
