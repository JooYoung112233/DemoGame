# 시스템: 길드 평판 (Guild Reputation)

> 길드 전체의 사회적 위상. 0-100. NPC 거래, 마을 이벤트, 후반 콘텐츠 해금에 영향.

## 1. 평판 누적

### 1.1 획득원

| 행동 | 평판 |
|---|---|
| 메인 도전 성공 (Lv 클리어) | +0.5 |
| 메인 도전 실패 | -0.2 |
| 보스 처치 | +5 |
| NPC 호감도 1티어 상승 | +2 |
| NPC 거래 (1회당) | +0.2 |
| 마을 이벤트 긍정 선택 | +1~5 |
| 마을 이벤트 부정 선택 | -1~3 |
| 용병 사망 (런 중) | -1 |

### 1.2 가속

- 길드회관 E7: 평판 획득 +30%
- 길드회관 E12: 모든 평판 변화 정보 공개

## 2. 마일스톤

| 평판 | 타이틀 | 해금 콘텐츠 |
|---|---|---|
| 0 | 무명 길드 | 기본 |
| 15 | 이름 있는 길드 | 길드회관 7-9단계 게이트 통과, 일반 의뢰 |
| 35 | 신뢰받는 길드 | 길드회관 10-12단계 게이트, NPC 특수 의뢰 |
| 60 | 전설의 길드 | 영주 직속 의뢰, 마을 이벤트 분기 추가 |
| 100 | 왕실 인정 | 엔드게임 콘텐츠, 왕실 의뢰 (전설급 보상) |

## 3. 마일스톤별 해금 상세

### 평판 15 — 이름 있는 길드
- NPC 매물 슬롯 +1 (8명 전체)
- 일반 의뢰 게시판 등장 (마을)
- 마을 이벤트 빈도 +10%
- 길드회관 단계 7-9 해금 (가격은 그대로)

### 평판 35 — 신뢰받는 길드
- NPC 특수 의뢰 (호감도와 별도)
- 길드회관 단계 10-12 해금
- 영주의 사절 등장 (마을 이벤트)
- 모집소 희귀 등급 확률 +5%

### 평판 60 — 전설의 길드
- 영주 직속 의뢰 (큰 보상, 특수 조건)
- 마을 이벤트 분기 추가 (정치/외교 라인)
- 모집소 에픽 등급 확률 +3%
- 자동 공급 계약 (희귀 소재 정기 입수)

### 평판 100 — 왕실 인정
- 왕실 의뢰 (전설 등급 의뢰, 전설 보상)
- 모집소 전설 등급 확률 +2%
- 길드회관 8단계 (마지막) 게이트 통과 시 추가 보너스
- 엔드게임 이벤트 체인 (시나리오)

## 4. 평판 하락 콘텐츠

평판은 하락도 가능. 음의 효과:

| 평판 | 효과 |
|---|---|
| 음수 ~ 0 | 변화 없음 (최저 0) |
| 0-15 | 일반 의뢰 봉인 |
| 의도적 -5 | NPC 일부 호감도 -10 |
| 의도적 -10 | NPC 거래 거부 (일시) |

평판은 절대 안 떨어지진 않음 (이벤트 결과로만 하락). 일반 플레이로 -효과 거의 안 봄.

## 5. UI 표시

```
TownScene HUD 좌상단:
  ⭐ 길드 평판: 42 (신뢰받는 길드)
  다음 마일스톤: 60 / 18 남음
```

- 평판 변화 시 플로팅 +/- 표시
- 마일스톤 도달 시 모달 (해금 콘텐츠 안내)

## 6. gameState

```javascript
guildReputation: 0  // 0-100
```

기존 [game-state.md](game-state.md) 필드 그대로 사용.

## 7. 시스템 연동

- **길드회관**: 단계 7+/10+ 게이트 (각 15/35 평판 요구)
- **NPC 호감도**: 호감도 상승이 평판 가속, 평판이 NPC 특수 의뢰 해금
- **마을 이벤트**: 이벤트 선택지 결과의 핵심 변수
- **모집소**: 평판 높을수록 좋은 등급 등장
- **엔드게임**: 평판 100 도달이 시나리오 트리거
