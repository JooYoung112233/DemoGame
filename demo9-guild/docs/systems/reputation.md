## 1. 정의

```
길드 평판 (0-100)

획득원:
  - 메인 도전 성공: +0.5
  - 보스 처치: +5
  - NPC 호감도 누적: +0.2 / 거래
  - 마을 이벤트 (선택지에 따라): -3 ~ +5
  - 길드회관 E7, E18: 가속 (E7: +30%, E12: 모든 정보 공개)

마일스톤:
  15: "이름 있는 길드"
  35: "신뢰받는 길드"
  60: "전설의 길드"
  100: "왕실 인정"

용도:
  - 길드회관 게이트 (단계 7-9: 평판 15, 단계 10-12: 평판 35)
  - NPC 특수 의뢰 해금
  - 마을 이벤트 분기
```
