# 시스템: 장비 + 자동화

## 1. 장비 슬롯 (4슬롯 + 파티 공용 1)

### 1.1 용병 개인 슬롯

| 슬롯 | 주 효과 |
|---|---|
| weapon | ATK, CRIT |
| armor | DEF, HP |
| accessory | 특수 (CRIT/SPD/특성 강화) |
| consumable (1회용) | 전투 시작 시 자동 소비 (포션 등) |

### 1.2 파티 공용 — pocket (수동 아이템)

> v4 추가. BP 전용 (다른 구역에서는 슬롯 자체는 보이되 사용 불가).

| 슬롯 | 주 효과 |
|---|---|
| pocket × 2 (파티 공용) | BP 쉬는 곳에서 **수동 사용** (자동 X) |

- 해금: 길드회관 F2
- 자세한 사용 규칙·아이템 풀: [zone-bloodpit.md §8](zone-bloodpit.md)
- consumable 슬롯과 분리(서로 영향 없음)

## 2. 구역 특수 장비

각 구역에서만 드롭되는 전용 장비. 기본 스탯 + 긍정/부정 트레이드오프 + 구역 전용 특수 스탯.

### 2.1 구역별 테마

| 구역 | 접두어 | 특수 스탯 | 효과 |
|---|---|---|---|
| Blood Pit | 피에 물든 ~ | **출혈** | 공격 시 DoT 부여 (지속 피해) |
| Cargo | 증기 ~ | **과열** | 공격 누적 → 일정 수 도달 시 버스트 폭발 피해 |
| Blackout | 저주받은 ~ | **공포** | 공격 시 적 일시 행동불가 확률 |

### 2.2 트레이드오프

모든 구역 장비는 긍정 + 부정 스탯을 동시에 가짐.

| 예시 | 긍정 | 부정 |
|---|---|---|
| 피에 물든 철검 | ATK +15, 출혈 부여 | DEF -3 |
| 증기 갑옷 | DEF +8, 과열 누적 +10% | SPD -5% |
| 저주받은 반지 | 공포 15%, CRIT +5% | HP -8% |

### 2.3 구역 보너스

해당 구역에서 사용 시 추가 효과 발동:
- 피에 물든 장비 → BP에서 출혈 데미지 +50%
- 증기 장비 → Cargo에서 과열 폭발 데미지 +50%
- 저주받은 장비 → BO에서 공포 확률 +10%

### 2.4 장비 타입별 특수 스탯 차이

| 슬롯 | BP (출혈) | Cargo (과열) | BO (공포) |
|---|---|---|---|
| weapon | 출혈 DoT 부여 | 과열 누적 가속 | 공포 확률 |
| armor | 피격 시 출혈 반사 | 과열 시 방어 버프 | 공포 시 회피 버프 |
| accessory | 출혈 대상 추가 피해 | 과열 폭발 범위↑ | 공포 지속시간↑ |

## 3. 자동 장착 모드

- 클래스 추천 (warrior=HP/DEF, mage=ATK/CRIT...)
- 최고 ATK 우선
- 최고 HP+DEF 우선
- 균형
- 수동

**발동 시점 (둘 다 길드회관 D 노드 해금 필요):**
- TownScene 자동화 탭 "전체 최적화" 버튼 (D1 해금) — 사용자가 클릭한 순간 적용
- 파견 출발 직전 자동 적용 (D4 해금, 체크박스로 ON/OFF)

> ❌ **제거된 트리거 — "새 장비 획득 즉시 자동 장착"**
> 이유: 사용자가 새 장비 확인하기 전에 슬롯이 바뀌면 혼란. D1 수동 버튼 + D4 출발시 자동으로 충분 → 즉시 자동은 게임성 손해 대비 편의 이득 미미. (구 길드회관 D3 제거됨)

## 4. 장비 잠금

자물쇠 표시 → 자동 처분/교체에서 제외.

## 5. 자동 처분

> 설정 위치: **TownScene → '자동화' 탭** (길드회관 D2 해금 필요)

규칙 (체크박스):
- common 자동 판매
- uncommon 자동 판매
- 잠금 안된 중복 자동 판매
- 보관함 80% 차면 common부터

처분 방식: 즉시 판매 (70% 가치) 또는 위탁 등록 (D6 해금 시 위탁 자동).
