### 3.2 Cargo 웨이브 (4-7구간, Lv별)

```
Lv 1-3: 4구간 — 약탈자 위주, 산적 두목 1회
Lv 4-6: 5구간 — 산적 두목 다수, 폭풍 정령 등장
Lv 7-9: 6구간 — 기계 골렘 등장, 친위대 호위
Lv 10:  7구간 — 마지막 구간 화물 제독 (보스)
```

(구체적 적 수 / 라운드별 구성은 통합 시뮬 시 밸런싱)

### 3.3 Blackout 웨이브 (그리드 방 단위)

```
각 전투 방: 적 3-5체 (그리드 레벨에 따라)
Lv 1-3:  잔재 + 망령 위주
Lv 4-6:  골렘 + 함정 마스터 등장
Lv 7-9:  저주 사제 + 그림자 분신 등장
Lv 10:   보스방 = 군주의 그림자 호위 + 그림자 군주
```

## 4. 적 스탯 스케일링

```javascript
function getEnemyStats(baseStats, zoneLevel) {
  return {
    hp:  baseStats.hp  * (1 + (zoneLevel - 1) * 0.25),
    atk: baseStats.atk * (1 + (zoneLevel - 1) * 0.20),
    def: baseStats.def + (zoneLevel - 1) * 2
  };
}
```

## 5. 특수 능력 (적별)

### Blood Pit
- 광전사: HP 30%↓ 시 ATK +50%
- 핏 야수: 공격 시 출혈 부여 (5초 DoT)
- 처형인: HP 30%↓ 적에게 일격사 시도 (10초 쿨다운)

### Cargo
- 폭탄 약탈자: 칸 옆에서 자폭 (잡기 전 처치 필요)
- 산적 두목: 처치 전까지 약탈자 무한 소환
- 폭풍 정령: 5초마다 전체 화면 5% 데미지
- 기계 골렘: 일반공격 면역, 스킬만 통함

### Blackout
- 그림자 잔재: 50% 회피
- 저주받은 망령: 공격 시 저주 +1 누적
- 함정 마스터: 방 진입 시 함정 (정신 디버프)
- 그림자 분신: 처치 시 2개로 분열 (HP 50%씩)
- 저주 사제: 모든 아군 -10% 모든 스탯 디버프

## 6. 드롭 테이블 (구역×레벨)

| 등급 | 일반몹 | 엘리트 | 호위병 |
|---|---|---|---|
| common | 60% | 30% | 10% |
| uncommon | 30% | 40% | 30% |

구역 특수 장비(피/증기/저주)는 엘리트+ 드롭에서 추가 확률.

## 7. 호위병 (Lv 4-10 강화)

호위병은 Lv 4부터 등장하며 라운드 후반부 또는 보스방 주변에 배치.

| 구역 호위병 | Lv4-6 | Lv7-9 | Lv10 (보스 동반) |
|---|---|---|---|
| BP 핏 챔피언 | HP×1.5 | HP×2 | HP×2.5 |
| Cargo 친위대 | HP×1.5 | HP×2 | HP×2.5 |
| BO 군주의 그림자 | HP×1.3 | HP×1.8 | HP×2.2 |

호위병은 처치 시 보장 드롭 (rare+).
