# 구역 전투: Blackout (저주받은 저택)

> 비선형 탐색형 전투. 마인스위퍼식 타일 뒤집기 + 횃불 자원 관리 + 자발적 저주 트랙.
>
> **⚠ 전투 엔진 재설계 진행 중 (2026-05)**: BP/Cargo와 차별화된 "특이한 개성있는" 전투를 위해
> 다키스트 엔진(BP 기반)을 버리고 Blackout 전용 전투 엔진을 새로 설계 중. 3개 프로토타입을
> 동시 구현해 직접 플레이로 재미 검증 → 1개 선정 → 정식 통합 예정.
> 상세는 §0. 전투 엔진 프로토타입 (탐색은 본 문서 §2~ 그대로 유지)

## 0. 전투 엔진 프로토타입 (3개 동시 구현)

데모 단계에서 전투 자체의 개성을 검증하기 위해 3개 후보를 모두 구현. 탐색 시스템과는 분리된
독립 씬으로, 사용자가 메뉴에서 골라 플레이 → 가장 재미있는 방향을 정식 채택.

### 0.1 진입 플로우

```
DeployScene → 구역=Blackout 선택 → 출발
→ BlackoutProtoSelectScene (3개 카드 메뉴)
→ Grid / Lane / Rune 중 하나 선택 → 해당 전투 씬
→ 승리/패배 → 메뉴로 복귀 (또는 TownScene)
```

### 0.2 후보 1: 그리드 통합 전투 (`BlackoutGridScene`)

**컨셉**: 마인스위퍼 그리드 자체가 전투장. 탐색과 전투가 한 시스템.

| 항목 | 값 |
|---|---|
| 그리드 | 5×5 셀 (셀 크기 90px) |
| 아군 | 4명, 좌측 2열 배치 |
| 적 | 2~4마리 (구역 레벨 기반), 우측 2열 |
| 턴 진행 | SPD 순 1유닛씩 행동 |
| 행동 | 이동(인접) / 공격(사거리) / 스킬(쿨3R) / 횃불 패스(인접 아군) / 패스 |
| 시야 | 횃불 든 아군 인접 lightRadius 셀 = 빛 |
| 어둠 셀 적 | "?" 로만 표시, 인접 위험 숫자 (마인스위퍼) |
| 공격 제한 | 빛 안의 적만 직접 공격 가능 |
| 승리 조건 | 모든 적 처치 |

**클래스 차별**:

| 클래스 | 사거리 | 시야 | 특수 |
|---|---|---|---|
| warrior | 1 (인접) | 1 | HP/ATK 강함, 스킬: 강격+DEF 버프 |
| rogue | 1 | 1 | 어둠 셀에서 ATK ×1.5, 스킬: ×2.5 단일 |
| mage | 3 (8방향) | 1 | 스킬: 3×3 AoE 폭발 |
| archer | 4 (직선만) | 2 | 시야 확장, 스킬: 직선 관통 |
| priest | 2 (8방향) | 2 | 빛 범위 확장, 스킬: 인접 아군 50% 회복 |
| alchemist | 2 | 1 | 스킬: 3×3 폭탄 |

**핵심 결정 포인트**:
- 누가 횃불 들고 누가 어둠에 숨을지 — 매 턴 횃불 패스
- 어둠 셀 적의 위치를 위험 숫자로 추리
- 도적은 어둠에 머물러야 강함 ↔ 사제는 빛에 있어야 시야 확장

### 0.3 후보 2: 빛/어둠 듀얼 트랙 전투 (`BlackoutLaneScene`)

**컨셉**: 두 트랙(빛/어둠). 빛은 표적, 어둠은 잠복. 트랙 전환과 차징의 게임.

| 항목 | 값 |
|---|---|
| 트랙 | 2개 (☀ 빛 / 🌑 어둠) |
| 아군 | 4명 시작 시 분산 (홀짝) |
| 적 | 2~4마리, 분산 |
| 행동 | 트랙 전환 / 공격(빛만) / 스킬 / 차징(다음 ×2) / 패스 |
| 빛 트랙 | 공격 가능, 적이 우선 표적 |
| 어둠 트랙 | 공격 불가(예외 있음), 적이 못 봄, 차징 보너스 |
| 적 AI | 빛 아군 HP 낮은 순 공격, 빛 없으면 어둠 추격 (50% 확률) |

**클래스 적성**:

| 클래스 | 빛 보너스 | 어둠 보너스 |
|---|---|---|
| warrior | DEF +30% | - |
| rogue | - | 차징 시 ×3.0 (그림자 암살) |
| mage | - | 어둠에서 공격 50% 가능 (트랙 무시) |
| archer | - | 차징 시 일렬 관통 |
| priest | 회복 ×1.5 | - |
| alchemist | - | 폭탄 = 양 트랙 동시 데미지 |

**핵심 결정 포인트**:
- 빛에 누가 서서 도발 / 어둠에서 누가 차징
- 차징 후 한 방이냐, 연타냐
- 클래스 적성이 트랙 선택을 강제하지 않고 유도

### 0.4 후보 3: 봉인 의식 전투 (`BlackoutRuneScene`)

**컨셉**: HP를 깎지 않음. 적 룬 시퀀스를 순서대로 입력해 봉인.

| 항목 | 값 |
|---|---|
| 적 | 1~2마리 (보스급), 각각 3~5 룬 시퀀스 |
| 횃불 카운트 | 12라운드 시작, 매 라운드 -1, 0이면 룬 안 보임 → 패배 |
| 행동 | 라운드당 모든 아군 1회씩 룬 입력 (선택→룬 클릭) |
| 정답 | 적 진행도 +1, 콤보 +1 |
| 오답 | 적 카운터 (×2 데미지), 콤보 끊김 |
| 라운드 종료 | 모든 적이 1회 아군 공격, 횃불 -1 |
| 승리 | 모든 적 봉인 |
| 패배 | 횃불 0 또는 파티 전멸 |

**클래스 ↔ 입력 가능 룬**:

| 클래스 | 룬 풀 |
|---|---|
| warrior | 🛡 방패 |
| rogue | 🗡 단검, 🌑 어둠 |
| mage | 🔥 불, 💧 물, ❄ 얼음 (3종) |
| archer | 🎯 정밀 |
| priest | ✨ 정화, 🩸 봉인 |
| alchemist | 💧 물, 🔥 불 |

**핵심 결정 포인트**:
- 파티 클래스 조합이 곧 입력 가능 룬 폭 = 봉인 능력
- 콤보(2명 연속 정답) 시 보너스
- 횃불 줄어드는 압박 vs 신중한 입력
- 다양한 클래스 조합이 가치 있는 메타게임

### 0.5 비교 한눈에

| 항목 | Grid | Lane | Rune |
|---|---|---|---|
| 전투장 형태 | 2D 그리드 5×5 | 2 트랙 (1D) | 정적 정렬 |
| 데미지 모델 | HP 깎기 | HP 깎기 | 봉인 진행도 |
| 핵심 결정 | 위치 + 횃불 | 트랙 + 차징 | 룬 선택 + 콤보 |
| 시간 압박 | 없음 (라운드 무제한) | 없음 | 12 라운드 (횃불) |
| 마인스위퍼 정체성 | ⭐ 가장 강함 | 약함 | 없음 |
| 구현 무게 | 큼 | 중간 | 가장 가벼움 |
| 학습 곡선 | 가파름 | 보통 | 직관적 |
| 용병 개성 강도 | 위치 적성 ↑ | 트랙 적성 ↑ | 룬 풀 ↑ |

### 0.6 선정 후 통합 계획

플레이 결과로 1개 채택 후:
1. 채택한 엔진을 정식 `BlackoutBattleScene` 내 전투 처리에 통합 (기존 DarkestCombat 호출 제거)
2. 탐색 시스템 (§2~) + 새 전투 엔진 = 정식 Blackout
3. 미채택 2개는 보존(레퍼런스) 또는 archive 폴더로 이동
4. 친화도 트리/저주 트랙과 신규 엔진 통합 (예: Grid → `dark_eyes` 시야 확장)

## 1. 기본 구조

```
저택 입장 → 4×4 그리드 (방 16개, 전부 뒤집힘)
→ 횃불로 방 하나씩 공개 → 방 내용에 따라 전투/보물/이벤트/저주
→ 보스방 찾으면 도전 가능, 횃불 다 쓰면 강제 퇴각
→ 보스 클리어 = 성공
```

- **참여 인원**: 지정 파티 (BP처럼 편성)
- **전투 방식**: §0의 3개 프로토타입 중 선정 (다키스트 X). 일반공격 자동 + 스킬 수동 모델 폐기.
- **자동전투 잠금해제**: (보류) 새 엔진 확정 후 자동 모드 재설계

## 2. 그리드 시스템

### 2.1 맵 구조

| 구역 레벨 | 그리드 | 방 개수 | 횃불 |
|---|---|---|---|
| Lv 1-3 | 3×3 | 9 | 6 |
| Lv 4-6 | 4×4 | 16 | 9 |
| Lv 7-9 | 4×5 | 20 | 11 |
| Lv 10 | 5×5 | 25 | 13 |

- 시작 시 모든 방이 뒤집혀 있음 (내용 숨김)
- 입구는 공개된 상태, 인접 방만 클릭 가능
- 횃불 1개로 인접 방 1개 공개 + 진입
- 횃불 다 쓰면 강제 퇴각 (그때까지 획득물만 보존)

### 2.2 마인스위퍼식 힌트

- 방 공개 시 인접 8칸의 위험도 숫자 표시
- 예: "위험 2" = 인접 방에 함정/저주방 2개
- 보스방은 특수 표시 (예: 인접에 "👑 인접")
- 안전한 방(보물/이벤트)도 별도 힌트
- **어디를 열지** 판단이 핵심 게임플레이

## 3. 방 종류

### 3.1 일반 방 (기본 트랙)

| 종류 | 비중 | 효과 |
|---|---|---|
| 💀 전투 방 | 40% | 적 웨이브, 클리어 시 일반 보상 |
| 💰 보물 방 | 15% | 장비/골드 즉시 획득 |
| ❓ 이벤트 방 | 15% | 선택지 이벤트 (버프/디버프/거래) |
| 🔥 횃불 방 | 10% | 횃불 +2~3 회복 |
| 👑 보스 방 | 1개 고정 | 보스전. 클리어 = 런 성공 |
| 🚪 빈 방 | 19% | 별 거 없음 (그냥 통로) |

### 3.2 저주 방 (옵션 트랙 — 도파민 추구자용)

저주 방은 가끔 등장 (구역 레벨에 따라 1~3개). **들어갈지 선택**.

| 종류 | 효과 | 저주 |
|---|---|---|
| 🔮 저주 보물방 | 고급 장비 1개 (희귀+), 어려운 전투 | +1 |
| 🕯 저주 상점 | 저주 장비 판매상 | +1 |
| 🩸 저주 의식방 | 영구 버프 1개 (영혼 거래) | +2 |
| 🌑 봉인된 방 | 전설급 보상, 매우 어려운 전투 | +3 |
| 🕸 심연의 방 | 보스급 보상 (런 1회 한정) | +4 |

### 3.3 저주 트랙 해금

- 저주 방은 누적 저주 레벨에 따라 등장:
  - 저주 0: 🔮 저주 보물방, 🕯 저주 상점
  - 저주 2+: 🩸 저주 의식방 등장
  - 저주 5+: 🌑 봉인된 방 등장
  - 저주 8+: 🕸 심연의 방 등장
- 들어가지 않으면 저주 안 쌓임 → **일반 플레이어는 무시 가능**
- 들어가면 저주↑ but 보상 大

## 4. 저주 효과

저주 레벨이 올라갈수록:

| 저주 | 적 강화 | 추가 효과 | 해금 콘텐츠 |
|---|---|---|---|
| 0 | - | - | 일반 방만 |
| 1-2 | +10% | 함정 데미지 +10% | 🩸 의식방 |
| 3-4 | +20% | 저주 장비 효과 +20% | 🌑 봉인된 방 |
| 5-6 | +35% | 저주 장비 효과 +40% | 🕸 심연의 방 |
| 7-8 | +50% | 저주 장비 효과 +60%, 일반 보상도 +20% | 보스 강화 |
| 9-10 | +75% | 저주 장비 효과 2배 | 보스 페이즈 추가 |

**핵심**: 저주받은 장비를 끼고 일부러 저주를 쌓으면 폭발적 강화. 저주 추구 빌드.

## 5. 횃불 (탐색 자원)

- 시작 횃불은 그리드 크기에 비례
- 사용처:
  - 인접 방 공개 + 진입: 1개
  - 같은 방 재진입 (보물/이벤트 재방문): 0개 (자유 이동)
  - 멀리 떨어진 방 워프: 2개 (상위 친화도 노드)
- 회복:
  - 🔥 횃불 방: +2~3
  - 이벤트 선택지 결과로 가끔 +1
  - 친화도 트리 노드 `dark_torch` 보유 시 +1

## 6. 보스 도전

- 👑 보스방 위치는 그리드 안에 1개 (랜덤 배치)
- 보스방 도달 후:
  - 즉시 도전 가능
  - 또는 더 탐색 후 도전 (보스방 위치는 기억됨)
- 보스 강화: 저주 레벨에 비례 → 저주 0이면 기본, 저주 10이면 매우 강함
- 보스 보상: 저주 레벨 비례 → 저주 높을수록 더 좋은 드롭

## 7. 길드회관 H(어둠 통제) 연동

- H3: 횃불 시작량 +2
- H6: 마인스위퍼 힌트에 안전한 방도 표시
- H9: 그리드에서 워프 가능 (멀리 떨어진 방으로)
- H12: 보스방 위치 처음부터 공개

## 8. 친화도 트리 연동

- `dark_torch` (Lv1): 시야 +1
- `shadow_step` (Lv2): 이동속도 +10%
- `curse_bearer` (Lv3A): 저주 Lv당 ATK +3% → 저주 추구 빌드 핵심
- `purifier` (Lv3B): 저주 상승 속도 -25%
- `curse_master` (CA): 저주 효과 양성 전환 → 저주 빌드 캡스톤

## 9. 서브 파견 vs 메인 전투

| | 메인 전투 | 서브 파견 |
|---|---|---|
| 그리드 탐색 | 수동 (방 선택) | 자동 (시뮬) |
| 저주 방 | 선택 가능 | 자동 회피 |
| 횃불 관리 | 플레이어 | 시뮬 |
| 스킬 | 수동 발동 | 없음 |

서브 파견은 저주 트랙 못 탐 → 저주 장비/도파민 보상은 메인 전용.
