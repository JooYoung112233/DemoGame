# 시스템: 경매장 (Auction)

> 골든이 진행하는 마을 시설. **2단계 미니게임 + 위작/저주 도박** 구조.
> 한 사이클 2~3분, 시간 토큰 2칸 + 입장료 소모. **골드 싱크** (장기 평균 마이너스, 운/실력으로 이득 노림).
> 다키스트 던전 톤의 도파민 콘텐츠 — 비공개 입찰의 눈치게임 + 저주받은 양면 아이템의 도박.

## 1. 디자인 핵심

| 축 | 선택 | 이유 |
|---|---|---|
| 방향성 | B+C 안 (2단계 + 위작/저주) | 살때 한 번, 팔때 한 번, 가끔 도박. 능동성 ↑ |
| 무게 | 식사형 (2~3분, 토큰 2칸) | 다른 시설(BP/Cargo) 사이 끼우기 좋음 |
| 1단계 (살때) | 비공개 일회 입찰 (High Society형) | 눈치게임. NPC들과 동시 적어내고 공개 |
| 2단계 (팔때) | 가격 책정 + 손님 반응 (Recettear형) | 대사/표정으로 시세 학습. 캐릭터성 ↑ |
| 위작/저주 | 특정 조건 트리거 등장 | 상시 도박이 아니라 페이스의 변주 |
| 골드 흐름 | **싱크** (소모원) | 운/실력 없으면 손해. 전투 보상이 주 수입원 유지 |

## 2. 해금

- 길드 Lv 3, 건설 비용 800G
- 골든 NPC 동시 해금
- (선행 요건: 첫 BP 클리어 권장 — 매물 풀에 BP 드랍 등장)

## 3. 한 사이클 개요 (2~3분)

```
[경매장 입장]
  └─ 시간 토큰 2칸 소모
  └─ 입장료 50~100G 차감 (회당 고정)
  └─ 라운드 예산 결정 (보유 골드의 X%, 상한)
  
[1단계 — 비공개 일회 입찰] (40~60초)
  └─ 매물 3장 동시 노출 (희귀도/유형/시세 표시)
  └─ NPC 입찰자 1~3명 (매물별 동적, 시세 자동 산출)
  └─ 플레이어: 각 매물에 입찰액 적어내기 (라운드 예산 안에서 분배)
  └─ 동시 공개 → 최고가 낙찰 (동률 시 NPC 우선)
  └─ 낙찰된 매물 = 보유 풀로 이동

[전환 연출] (~5초)
  └─ 손님들 입장 ("어디 좋은 물건 있나?")

[2단계 — 가격 책정 + 손님 반응] (60~90초)
  └─ 손님 1~3명 순차 입장 (손님당 10~20초 타이머)
  └─ 손님에게 보여줄 매물 선택 → 가격 책정
  └─ 손님 반응: 대사 + 표정 + (호감도 따라) 점진적 시세 노출
  └─ 가격 조정 1~2회 가능 (타이머 안에서)
  └─ 손님 만족 → 판매 / 거절 → 다음 손님

[정산]
  └─ 1단계 지출 vs 2단계 수입 차액 표시
  └─ 골든 호감도 누적 (참가 +0.5, 만족 판매 추가 +1)
  └─ 안 팔린 매물 → 보관함 (수동 처분 가능)
```

## 4. 1단계 — 비공개 일회 입찰

### 4.1 매물 (3장)

- 라운드마다 무작위 3장 등장. **고소도 선택** (전부 사면 예산 폭주, 1~2장만 노리는 결단)
- 매물 정보 표시:
  - 아이콘, 이름, 희귀도
  - 카테고리 (무기/방어구/소비/소재/저주 등)
  - **추정 시세** (지점 가격, 호감도 따라 정확도 ↑)
  - NPC 입찰자 수 (1~3, 매물 등급별 동적)
  - 위작/저주 의심 표시 (감식 결과, §6 참조)

### 4.2 NPC 입찰자 (1~3명, 동적)

- 매물 등급/유형으로 동적 산출:
  - 흔한 매물: 평균 1명 (가끔 0명 → 헐값 가능)
  - 희귀: 평균 2명
  - 전설/특수: 평균 3명
- 골든 호감도가 높을수록 NPC 수 감소 (귀띔으로 일부 NPC가 빠짐 — 후술)

### 4.3 NPC 입찰액 분포 (매물 등급별)

| 매물 등급 | 분포 | 평균 | 특성 |
|---|---|---|---|
| common/uncommon | 좁은 정규 (시세 ±15%) | ≈ 시세 0.95x | 예측 쉬움, 박리다매 |
| rare | 정규 (시세 ±25%) | ≈ 시세 1.05x | 약간의 변동 |
| epic | 넓은 정규 (시세 ±40%) | ≈ 시세 1.15x | 큰 변동 — 운빨 ↑ |
| legendary | 매우 넓은 정규 (시세 ±60%) | ≈ 시세 1.30x | 폭등/폭락 가능. 도박성 |
| 저주/위작 (의심) | 50% 확률로 NPC 1명 빠짐 | 시세 0.7x | 헐값 함정 (위작이면 손해) |

- 시세는 가치(item.value) × marketModifier (구역 이벤트로 변동)
- 일부 매물은 일정 확률(예: 20%)로 NPC가 아예 입찰 안 함 → 헐값으로 잡을 기회

### 4.4 라운드 예산 (입찰 상한)

- **라운드당 총액 상한 = 보유 골드의 X% (조정 가능)** 으로 경매 예산 산정
- 3장 매물에 예산을 분배 (한 매물에 몰빵 가능)
- 매물당 최소 입찰액 = 시세 × 0.5 (입찰하지 않을 자유 있음)
- 매물당 최대 입찰액 = 시세 × 2.0 (익스플로이트 방지)
- 예산 부족하면 일부 매물에 입찰 불가

### 4.5 결과 처리

- 동시 공개 → 매물별 최고가 낙찰
- 동률 시 NPC 우선 (플레이어 약간 불리한 룰 — 도박성 ↑)
- 낙찰 실패 매물: 골드 환급, 보관함으로 안 감
- 낙찰 성공 매물: 골드 차감, 보유 풀로 이동 (2단계 판매 대상)

## 5. 2단계 — 가격 책정 + 손님 반응

### 5.1 손님 NPC 풀 (혼합 구성)

- **고정 로테이션 4~6명** (이름/실루엣 + 선호 카테고리)
  - 예시 슬롯 (이름/외양/선호는 별도 결정 — §13 TODO)
- **귀임 손님** — 이벤트성 1~2명 (희귀 카테고리 선호, 가격대 ↑)
- 한 라운드에 등장 = 풀 중 1~3명 (호감도/조건 따라)

### 5.2 가격 책정 UX (혼합: 대사 + 표정 + 점진적 시세 노출)

- 매물을 손님에게 제시 → 가격 입력
- 손님 반응:
  - **대사**: "터무니없군", "흥…", "그 정도라면…", "당장 사겠소!"
  - **표정**: 😠 / 😐 / 🙂 / 😍 (4단계)
  - **시세 힌트** (점진적 노출):
    - 호감도 0~30: 표정만
    - 호감도 30~60: + 대사 (방향성 힌트)
    - 호감도 60~100: + 시세 화살표 (↑↑/↑/=/↓/↓↓)
    - 골든 호감도 4단계: + 정확한 손님 만족가 표시 (옵션)
- 가격 조정 1~2회 가능 (타이머 안에서)
- 손님이 거절(대사로 가버림) 또는 만족(아이콘 강조 후 매각)

### 5.3 판매 시간 압박

- **손님당 타이머 10~20초** (매물 등급/손님 까다로움 따라 다름)
- 타이머 소진 시 자동 거절 (안 팔린 매물은 보관함으로)
- 매물 등급↑ = 타이머↑ (전설 매물은 신중히 결정할 시간 줌)

### 5.4 손님 선호 노출 (학습형)

- 처음에는 모두 ???
- N회 거래(만족/거절 모두)하면 자동 메모 추가:
  - 거래 3회: 선호 카테고리 1개 드러남 ("무기 좋아함")
  - 거래 7회: 선호 카테고리 + 가격대 ("비싼 무기 선호")
  - 거래 15회: 추가 선호 + 싫어함 ("저주는 사절")
- 골든 의뢰 보상으로 가속 가능 (단계별)

## 6. 위작 / 저주 (C 요소)

### 6.1 등장 빈도 — 특정 조건 트리거

상시 등장이 아니라 페이스의 변주.
| 트리거 | 효과 |
|---|---|
| 구역 보스 처치 후 24h | 해당 구역 매물 풀에 위작/저주 +1 슬롯 |
| 평판 마일스톤 도달 | 다음 N회 경매에 전설 위작 1회 등장 가능 |
| 그림자(암시장) 호감도 30+ | 저주 매물 등장 확률 상시 +10% |
| 마을 이벤트 "암시장의 손길" | 다음 경매 위작 확률 25% |
| 일반 상태 | 등장 확률 0% (절대 안전) |

→ 트리거 발동 중일 때만 매물에 위작/저주 의심 가능. 톤은 "이벤트성 긴장".

### 6.2 감식 (자동 + 능동 혼합)

#### 자동 감식 (골든 호감도 → 단계 해금)

| 골든 단계 | 자동 경고 |
|---|---|
| 0~1 단계 | 없음 (트리거 발동 중에도 표시 X) |
| 2 단계 | "이상한 기운이 느껴진다" (위작/저주 구분 불가) |
| 3 단계 | "위작 의심" / "저주 의심" 구분 표시 |
| 4 단계 | 위작 100% 적발 + 저주는 양/음 효과 1줄 미리보기 |

#### 능동 감식 (저비용 상시 가능)

- 매물 클릭 → [감식] 버튼
- 비용: 매물당 골드 X (단계와 무관하게 적은 비용으로 계속 가능)
- 결과: 자동 경고 대비 1단계 위 결과 노출 (예: 자동 2단계면 능동 시 3단계 결과)
- 시간 토큰 소모 없음 (입찰 타이머 안에서 결정)
- 호감도 4단계 도달 후엔 능동 감식 무료

### 6.3 위작 vs 저주 — 페널티 차등

| 종류 | 외형 | 정체 | 페널티 |
|---|---|---|---|
| **위작** | 진품과 동일 | 그냥 가짜. 효과 무 / 가치 0 | **단순 손해**. 판매 시 시세 10~20%만 회수. 손님도 안 속음 (가격 폭락) |
| **저주** | 진품과 유사 | 진짜 효과 + 음성 페어 | **양면 아이템** — 강한 양성 + 강한 음성. 다키스트 trinket 톤. 잘 쓰면 이득 |

저주 예시 (수치는 §13 TODO):
- "핏빛 단검": ATK +25% / 매 라운드 자해 5
- "굶주린 흉갑": DEF +30 / 회복 효율 -50%
- "속삭이는 반지": 크리율 +20% / 사망 시 파티원 1명 공포

저주는 사제(정화) 또는 그림자(저주 교환)로 해소/활용 가능. 일부 빌드에선 의도적 채용 가치 있음.

### 6.4 골든 호감도 → 감식 곡선 (단계 해금 + 수동 상시)

- 단계 보상 = 자동 경고 정확도 점프
- 능동 감식 = 단계 무관하게 적은 비용으로 항상 사용 가능 → 호감도 0이라도 신중하면 막을 수 있음
- 4단계 도달 시 능동 감식 무료 + 자동이 거의 완벽

## 7. 골든 NPC 의뢰 4단계 (신사양)

> 기존 [npc-merchants.md §3.4](npc-merchants.md) 골든 라인을 신메커닉에 맞춰 재정렬.
> 단계별 보상 슬롯만 정의, **구체 의뢰 텍스트/수치는 TODO** (§13).

| 단계 | 호감도 | 의뢰 슬롯 | 보상 슬롯 |
|---|---|---|---|
| 1 | 10 | [경매 참가 횟수] | 입장료 -X% / 능동 감식 비용 -X% |
| 2 | 30 | [입찰 성공 횟수 또는 판매 만족 횟수] | **자동 감식 2단계** (이상한 기운) + 매물 미리보기 24h |
| 3 | 60 | [특정 등급 매물 거래 횟수] | **자동 감식 3단계** + **VIP 경매장 입장** (§8) |
| 4 | 100 | [골든의 신뢰 — 특별 의뢰] | **자동 감식 4단계** + 능동 감식 무료 + 손님 선호 영구 공개 |

→ 단계 진행 = 도박성↓ 안정성↑. 골든을 키우면 경매장이 점차 수익 가능한 구조로 변함.

## 8. VIP 경매장 (호감도 3단계 보상)

기본 경매장과 병렬 시설 (전환 가능). 차별점:

| 요소 | 일반 | VIP |
|---|---|---|
| 매물 풀 | 전등급 혼합 | **전설 등급 비중 ↑** (희귀 60% / 전설 35% / 기타 5%) |
| 손님 풀 | 일반 고정 4~6명 + 귀임 | + **VIP 손님** (높은 가격대, 까다로움) |
| 입장료 | 50~100G | 200~500G |
| NPC 입찰자 수 | 1~3 | 2~4 (경쟁 강함) |
| 위작/저주 확률 | 트리거 기반 | **+10%** (고급 매물일수록 함정 가능성↑) |
| 한 사이클 시간 | 2~3분 | 3~4분 (매물당 타이머 길어짐) |
| 골드 흐름 | 싱크 (마이너스) | 더 큰 싱크 (한 사이클 ±수천G) |

VIP는 "엔드게임 도박장" 포지션. 평소엔 일반 경매장 사용.

## 9. 경제 흐름 / 골드 싱크 정책

**경매장은 골드 싱크다.** 평균적으로 플레이어가 손해보는 구조.

### 9.1 싱크 메커니즘 4종

1. **입장료 (고정)** — 회당 50~100G. 미세하지만 누적.
2. **1단계 입찰가 vs 시세** — NPC 평균 입찰가가 시세보다 조금 높음 (rare 1.05x, epic 1.15x, legendary 1.30x). 낙찰하려면 평균보다 더 써야 함.
3. **2단계 판매가 vs 시세** — 손님 만족가는 시세보다 낮음 (평균 0.85~0.95x). 가격 책정 잘하면 시세 근처, 못하면 헐값.
4. **위작 손해** — 위작 사면 회수 10~20%만. 저주는 trinket 가치로 일부 회수.

→ 평균 사이클: -10~30% (시세 기준). 단 운/실력 좋으면 +50~100% 가능.

### 9.2 균형 가드

- 라운드 예산 상한 (보유 골드의 X%) → 한 번에 망하지 않음
- 매물당 최대 입찰액 2배 → 인플레 방지
- 안 팔린 매물 보관함 회수 → 완전 손실은 아님

### 9.3 골든 4단계 도달 시

- 자동 감식 + 무료 능동 감식 + 손님 선호 공개 → 위작/저주 회피 + 판매 효율↑
- 평균 사이클이 ±0 근처 (소액 이득 가능)
- 메인 수입원은 여전히 전투. 경매장은 보조 + 도박 + 장비 수집
