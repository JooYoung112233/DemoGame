# 시스템: 특성 제거 / 변환

> 음성 특성 정화 / 양성 특성 부여 / 전설 특성 부여 경로 정의.
> 통합 특성 테이블 자체는 [data/traits.js] + 추후 분리 문서로 이관 (TBD).

## 1. 음성 특성 제거

### 신전 — 정화 의식
- 음성 특성 1개 제거
- 비용: 길드Lv × 500G + 희귀도 배율
- 쿨다운: 용병당 5런

### 고급 모집소 — 특성 재추첨
- 음성 전체 재추첨 (다른 음성으로 바뀔 수도)
- 비용: 800G
- 확률: 30% 모두 제거 / 50% 1개 남음 / 20% 그대로

### Blackout 비밀방 — 정화의 샘
- 무작위 발견, 음성 1개 무료 제거

## 2. 양성 특성 추가

### 훈련소 — 특훈
- 양성 1개 부여 (무작위, 클래스 가중치)
- 비용: 1500G + 훈련 포인트 3
- 슬롯 제한: 최대 3개 양성

### 전설 특성 부여
- legendary 등급 + 80레벨 친화도 → 자동 부여
- 또는 보스 처치 후 토큰으로 부여

## 3. 특성 슬롯

- 양성: 최대 3슬롯
- 음성: 최대 2슬롯
- 전설: 1슬롯 (legendary만)
