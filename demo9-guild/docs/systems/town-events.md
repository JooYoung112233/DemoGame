# 시스템: 마을 이벤트 (NPC 의뢰 중심 재설계)

> 마을 이벤트의 핵심은 **NPC와의 관계 발전**. 단발 우발이 아닌 NPC 라인의 일부로 동작.
> NPC 의뢰 라인 자체는 [npc-merchants.md](npc-merchants.md) §3 참고.

## 1. 이벤트 분류 (재설계)

| 분류 | 비중 | 설명 |
|---|---|---|
| 🎯 **NPC 의뢰 라인** | 50% | 각 NPC의 4단계 의뢰 스토리 (32개 = 8 NPC × 4 단계) |
| 🤝 **NPC 관계 이벤트** | 20% | NPC 간 관계/대립/협력 (그림자 vs 사제 등) |
| 🏛 **평판 마일스톤 이벤트** | 15% | 평판 도달 시 발생하는 정치/세력 이벤트 |
| 🎲 **우발 이벤트** | 15% | 자연재해/방문자/사고 등 (NPC와 약하게 연관) |

## 2. NPC 의뢰 라인 (32개 이벤트)

> NPC 8명 × 4단계 = 32 이벤트. 호감도 진행에 따라 자동 등장.

### 발생 조건
- 해당 NPC 호감도가 단계 임계점 도달 (10/30/60/100)
- 이전 단계 완료 필수
- 첫 단계는 NPC 첫 방문 시 자동 트리거

### 이벤트 구조 예시

```
[크로그 1단계 — 거친 만남]

크로그가 큰 망치를 어깨에 메고 길드 문을 두드린다.
"새로 온 길드라지? 거 참 듬직해 보이는구만.
 가죽이 좀 필요한데, 한 5개만 가져다 줄 수 있겠나?
 후한 값을 쳐주리다."

[수락] BP 가죽 5개 의뢰 등록 → 완료 시 200G + 호감 +5
[거절] 호감 -2
[협상] (호감 5+) 선금 100G 즉시 받기, 완료 시 150G + 호감 +3
```

각 NPC 4단계 의뢰 구체적 내용은 [npc-merchants.md](npc-merchants.md) §3.

## 3. NPC 관계 이벤트 (8개)

NPC 간 상호작용에서 발생. 한쪽 편 들기 = 다른 쪽 호감 영향.

### 3.1 그림자 vs 사제 (대립)

#### "어둠의 유혹 vs 정화의 권유"
- **트리거**: 그림자 호감 30+ AND 사제 호감 30+
- **이벤트**: 그림자가 저주 의식 제안 / 사제가 만류
- **선택지**:
  - 그림자 편: 저주 장비 1개, 그림자 +10, 사제 -10
  - 사제 편: 정화 무료권, 사제 +10, 그림자 -10
  - 중립: 둘 다 -3

### 3.2 크로그 ↔ 헤르타 (협력)

#### "합작 의뢰 — 풀세트"
- **트리거**: 크로그 호감 30+ AND 헤르타 호감 30+
- **이벤트**: 둘이 합작해서 풀세트 제작 제안
- **선택지**:
  - 수락: 강철 10 + 가죽 10 → uncommon 풀세트 (무기+갑옷+장식구) 1세트
  - 거절: 둘 다 -2

### 3.3 골든 ↔ 미라 (협력)

#### "경매장의 비약"
- **트리거**: 골든 호감 30+ AND 미라 호감 30+
- **이벤트**: 경매장에 미라의 특수 비약 출품
- **선택지**:
  - 입찰: 1000G 베팅, 50% 영구 양성 1개 / 50% 잡템
  - 무시

### 3.4 ~ 3.8 (5개 추가)

- 스피드카 ↔ 그림자 (정보 거래)
- 아이언 ↔ 크로그 (강화 합작)
- 사제 ↔ 미라 (정화 vs 합성)
- 그림자 ↔ 스피드카 (음모)
- 헤르타 ↔ 사제 (성스러운 갑옷)

## 4. 평판 마일스톤 이벤트 (10개)

### 평판 15 도달
1. "이름 있는 길드의 첫 손님" — 마을 부유층 방문, 일반 의뢰 게시판 해금
2. "지방 의회의 인지" — 영주 사절 등장

### 평판 35 도달
3. "왕국의 시선" — 왕실 정보국 접근
4. "신뢰의 시험" — 정치 분쟁 중재 의뢰

### 평판 60 도달
5. "영주 직속 의뢰" — 큰 보상의 정치 의뢰
6. "경쟁 길드의 도발" — 외부 길드 대결

### 평판 100 도달
7. "왕실 인정 의식" — 왕실 접견 (엔드게임 트리거)
8. "전설의 길드" — 전 NPC 호감 +10

### 평판 하락
9. "악명의 그림자" — 평판 -10 도달, 그림자 호감 +15, 다른 NPC -5
10. "도적단의 접촉" — 음모 라인 진입

## 5. 우발 이벤트 (10개)

NPC와 약하게 연관된 외부/자연 이벤트. 게임 내 시간 30분마다 가중 추첨.

### 5.1 자연 재해 (4개)

1. **역병** — 마을 용병 30% 피로도 -20 / 미라 약초로 예방 (사제 호감 -2)
2. **폭풍** — 1시간 파견 시간 +25% / 일부 시설 폐쇄
3. **시장 변동** — 24시간 거래 가격 ±20% (랜덤)
4. **흉작** — 식료 가격 ×2 / 7일

### 5.2 외부 방문 (3개)

5. **떠돌이 상인** — 특수 매물 (희귀+ 보장), 비쌈
6. **모험가 길드 사절** — 협력 제안, 평판 +5
7. **다른 길드 도전** — 1:1 결투, 승리 시 보상/패배 시 손실

### 5.3 발견/사고 (3개)

8. **비밀 통로 발견** — 마을 지하 1회 탐색 (보상 풀)
9. **유적 단서** — 비밀 소문 1개
10. **길거리 카드 게임** — 도박 (100G ↔ 200G)

## 6. 길드원 사건 (별도 카테고리)

특정 용병에게 발생하는 이벤트. 음성 특성 / 사연 등.

| 트리거 | 이벤트 |
|---|---|
| 음성 특성 보유 용병 | "과거의 비밀" — 정화 기회 / 갈등 |
| 신참 (Lv 5↓) | "베테랑의 가르침" — 멘토 매칭 |
| 베테랑 (Lv 15+) | "유산 상속" — 영구 장비 / 의뢰 |
| 음성 `lone_wolf` | "고독한 늑대의 결심" — 양성 전환 기회 |
| 음성 `jealous` | "질투의 폭발" — 다른 용병과 갈등 |

총 ~10개 (개별 라인업 TBD)

## 7. 이벤트 구조 (코드)

```javascript
{
  key: 'krough_quest_1',
  category: 'npc_quest',
  npcId: 'krough',
  stage: 1,
  title: '거친 만남',
  body: '...',
  illustration: 'krough.png',
  conditions: {
    minFavor: { krough: 10 },
    notCompleted: 'krough_quest_1'
  },
  weight: 100,  // 의뢰 라인은 가중치 매우 높음 (조건 만족 시 즉시)
  choices: [
    {
      text: '수락',
      effects: { addQuest: 'krough_leather_5' },
      delayed: {
        questCompleteReward: { gold: 200, npcFavor: { krough: 5 } }
      }
    },
    {
      text: '거절',
      effects: { npcFavor: { krough: -2 } }
    },
    {
      text: '협상 (호감 5+)',
      conditions: { minFavor: { krough: 5 } },
      effects: { gold: 100, addQuest: 'krough_leather_5' },
      delayed: {
        questCompleteReward: { gold: 150, npcFavor: { krough: 3 } }
      }
    }
  ]
}
```

## 8. 발생 룰

### 8.1 우선순위

1. **NPC 의뢰 라인** (조건 만족 시 즉시) — 가중치 100
2. **NPC 관계 이벤트** (조건 만족 시) — 가중치 50
3. **평판 마일스톤** (도달 즉시) — 가중치 100
4. **우발 이벤트** — 가중치 10 (기본 풀)

### 8.2 빈도

- 마을 누적 접속 시간 30분마다 이벤트 1개 발동
- 길드회관 E3: -30% (20분마다)
- NPC 의뢰는 조건 만족 시 즉시 (위 30분 룰 무시)

### 8.3 반복 방지

- 최근 발생 이벤트 10개 가중치 ×0
- NPC 의뢰는 완료 후 다시 안 나옴 (단발)
- 관계/마일스톤 이벤트도 단발
- 우발 이벤트만 반복 가능 (가중치 페널티 있음)

## 9. 총 이벤트 수

| 카테고리 | 개수 |
|---|---|
| NPC 의뢰 라인 | 32 (8 NPC × 4 단계) |
| NPC 관계 이벤트 | 8 |
| 평판 마일스톤 | 10 |
| 우발 이벤트 | 10 |
| 길드원 사건 | ~10 |
| **합계** | **~70** |

목표 40개 초과 달성. 다만 NPC 의뢰 32개의 구체적 본문은 [npc-merchants.md](npc-merchants.md)에 의뢰 라인으로 정의되어 있음.
