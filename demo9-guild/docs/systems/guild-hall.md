# 시스템: 길드 회관 (Guild Hall) — v3 확정

> 메타 업그레이드의 척추. **8 카테고리 × 12 단계 = 96 업그레이드.**

## 1. 컨셉

게임 시작부터 보유. 모든 시스템의 **메타 업그레이드 트리**.

- 기존 시설 해금(제작소/경매장 등) = 수평 확장 (새 활동 추가)
- 길드 회관 = 수직 확장 (기존 활동 효율 강화)

## 2. 구조

| 카테고리 | 코드 | 컨셉 | 최대 단계 | 해금 조건 |
|---|---|---|---|---|
| A. 운영 | `operations` | 파견 슬롯, 메인 파티 인원 | 12 | 기본 |
| B. 인프라 | `infrastructure` | 로스터, 보관함, 보안 컨테이너 | 12 | 기본 |
| C. 휴식 | `recovery` | 스테미너 회복, 부상, 사망 구제 | 12 | 기본 |
| D. 자동화 | `automation` | 자동 장착/판매/회수/재파견 | **8** | 기본 |
| E. 정보 | `intel` | 미리보기, 소문, 평판 가속 | 12 | 기본 |
| F. 핏 통제 | `pit_control` | BP 핏게이지 메커니즘 변경 | 12 | BP 첫 클리어 |
| G. 화물 통제 | `cargo_control` | Cargo 열차 칸 메커니즘 변경 | 12 | Cargo 첫 클리어 |

> **D 자동화는 8단계 예외** — 자동화는 잡일 제거가 본질이라 핵심 기능만 추려 단축. 즉시장착/자동모집 같은 게임성을 갉아먹는 자동화는 제외. 그 외 카테고리는 모두 12단계.

## 3. 비용 구조

```
cost(stage) = round(baseCost × 1.5^(stage-1))

예시 (baseCost = 300):
  Lv1:   300G      Lv5:  1,519G     Lv9:   7,695G
  Lv2:   450G      Lv6:  2,278G     Lv10: 11,543G
  Lv3:   675G      Lv7:  3,418G     Lv11: 17,314G
  Lv4: 1,013G      Lv8:  5,127G     Lv12: 25,972G
```

## 8. D. 자동화 (Automation) — base 500G, **8단계**

> 모든 D 노드 해금 효과는 **TownScene "자동화" 탭**에서 토글/설정. 노드 해금 ≠ 자동 ON — 룰/체크박스 설정 필요.
> UI 명세는 [equipment-automation.md](equipment-automation.md) §6 참고.

| 단계 | 효과 |
|---|---|
| 1 | **자동 장착 해금** — 자동화 탭 "전체 최적화" 버튼 활성 (수동 클릭, 모드 선택: 클래스 추천/최고 ATK/HP+DEF/균형) |
| 2 | **자동 판매 룰 해금** — 자동화 탭의 자동판매 토글 사용 가능 (등급별 common/uncommon/rare중복, 보관함 80% 차면) |
| 3 | **자동 회수** — 파견 완료 시 보상/장비 즉시 처리 (수동 회수 모달 생략) |
| 4 | **파견 출발 자동 최적화** — 파견 출발 시 자동 장비 적용 체크박스 사용 가능 |
| 5 | **자동 재파견** — 같은 구역으로 자동 재출발 토글 (파견 슬롯 회전) |
| 6 | **자동 위탁** — 잡템 자동 위탁 등록 (경매장 연동, NPC 골든 호감도 무관) |
| 7 | **자동 치유/스테미너 관리** — 부상자 자동 부활(신전 골드 자동 차감), 마을 진입 시 피로 회복 시설 자동 이용 |
| 8 | **🏆 완전 자동화** (캡스톤) — 풀 자동 (장비 강화 + NPC 거래 + Cargo 카드 3택1 선택). 마을 잡일이 0이 됨 |

D 자동화는 8단계로 단축되어 누적 비용도 1/5 수준 (8단계: `500 × (1.5^8 - 1) / 0.5 ≈ 24,628G`). 자동화는 잡일 제거 척추라 비용 부담도 가볍게 → **자동화 우선 투자**가 자연스러운 빌드 옵션이 됨.

12시간 내 길드회관에 쓸 수 있는 골드 ≈ 200,000-300,000G
→ 2-3개 카테고리 풀업 or 모든 카테고리 7-8단계. D는 풀업 비용이 가벼우니 거의 모든 빌드에서 풀업 가능.
