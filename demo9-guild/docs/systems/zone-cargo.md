# 구역 전투: Cargo (기차 방어)

> 기차 내부 방어 + 라이트 RTS + 나락식 층 진행. 디아블로4 나락(The Pit) 구조.

## 1. 기본 구조

```
마을 → 층 선택 → 시작 모디파이어(양성 3택1 + 음성 강제 1) → 전투 시작
→ 5분 버티기 (적 침입 방어, XP 레벨업 카드 3택1)
→ 클리어 or 실패 → 마을 복귀
→ 성적에 따라 +1~3층 해금 → 반복 (최대 99층)
```

- **참여 인원**: 지정 파티 4명 (전원 동등, 친화도/시너지 적용)
- **전투 방식**: 라이트 RTS (4명 전원 클릭으로 이동/공격 대상 지정, 실시간)
- **1런**: 5분 고정
- **자동전투 잠금해제**: 해당 층 첫 클리어 후 자동 모드 해금 (AI 자동 조작 + 시각 관전)

## 2. 시점 / 공간

- **카메라**: 사이드뷰 단면도 (FTL식)
- **기차**: 단일 방 (칸 편성 폐기). 외벽 + 내부 공간
- **캐릭터**: 기차 내부에서 자유 이동 (RTS 명령)
- **적**: 외부에서 접근 → 외벽 파괴 → 내부 침입

## 3. 기차 방어

### 3.1 기차 HP (= 외벽 HP)

기차는 단일 HP 게이지를 가짐. 적이 기차 외벽에 도달하면 공격 시작.

```javascript
// 적이 외벽에 도달 시
function enemyAttackWall(enemy, train) {
  train.hp -= enemy.atk;
  if (train.hp <= 0) {
    // 런 실패
    endRun(false);
  }
}
```

### 3.2 2단계 침입

1. **외벽 단계**: 적이 기차에 도달 → 외벽 HP 감소
2. **내부 단계**: 외벽 HP 0 → 적이 내부 침입 → 캐릭터와 직접 전투

- 외벽이 깨지면 적이 내부로 들어옴
- ���부에서 적을 처치해도 외벽은 복구되지 않음 (카드/모디파이어로만 수리)
- **핵심**: 외벽이 깨지기 전에 적을 처치하는 것이 최선

### 3.3 클리어 / 실패 조건

| 조건 | 결과 |
|---|---|
| 5분 생존 (기차 HP > 0) | 클리어 |
| 기차 HP 0 | 실패 (마을 복귀, 같은 층 재도전) |

## 4. 나락식 층 진행

### 4.1 구조

- **최대 99층**, 1런 = 1층
- 클리어 시 성적에 따라 **+1 ~ +3층 해금**
- 마을에서 해금된 층 중 원하는 층 선택 도전
- 층 높을수록: 적 스탯 스케일링, 보상 품질 증가

### 4.2 성적 판정 (+1/+2/+3)

복합 기준 (기차 HP + 적 처치):

| 조건 | 해금 |
|---|---|
| 기차 HP 50%+ AND 적 전멸률 80%+ | +3층 |
| 둘 중 하나만 충족 | +2층 |
| 최소 클리어 (HP > 0, 5분 생존) | +1층 |

### 4.3 마일스톤

| 층 | 이벤트 |
|---|---|
| 10 | 엘리트 보스, 기본 클리어 보상 |
| 25 | 강화 보스, 희귀 보상 |
| 50 | 메가 보스, 전설 보상 |
| 99 | 최종 보스, 특별 보상 (업적) |

## 6. 카드 시스템 (뱀서식 레벨업)

### 6.1 메카닉

- 적 처치 → XP 누적 → 임계 도달 시 레벨업 → **3택1 카드 선택**
- 5분 런 내 약 **5~8회 레벨업** (초반 빠르고 후반 ���림)
- 카드 효과는 **런 한정** (마을 복귀 시 초기화)
- **3택1 중복 방지**: 한 선택지에 같은 카드 2장 이상 등장 불가
- **업그레이드 형태**: 하위 티어 보유 시 상위 티어 등장 가능

## 9. 친화도 트리 연동

Cargo 친화도 트리 노드:
- `iron_wall` (Lv1): 외벽 HP +5%
- `rapid_repair` (Lv2): 외벽 자동 수리 속도 +20%
- `pit_veteran` (Lv3A): 층 해금 시 항상 +1 추가
- `gambler` (Lv3B): 도박 카드 등장 시 양성 결과 확률 +15%
- `fortress_master` (CA): 외벽 파괴 불가 (대신 내부 직접 침입, 외벽 컨셉 변경)
