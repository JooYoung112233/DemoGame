# 구역 전투: Cargo (화물 열차)

> Cargo 전용 전투 메카닉. 열차 칸 운영 + 덱빌딩 + DPS 체크.

## 1. 기본 구조

```
칸 편성 → 출발 → 구간1(전투) → 클리어 시 카드 3택1 → 구간2(전투) → 카드 획득
→ 역 정차 [카드 활성화] → 구간3 → 카드 획득 → 구간4 → 역 정차 → ... → 종점 = 클리어
```

- **참여 인원**: 마을에 놀고 있는 전체 용병 (파견 중/강제 휴식 제외)
- **전투 방식**: 일반공격 자동 + 스킬 플레이어 직접 발동
- **자동전투 잠금해제**: 해당 레벨 첫 클리어 후 자동 모드 해금

## 2. 기차 칸 시스템

### 2.1 칸 편성

출발 전 보유 칸 중 선택해서 연결. 칸 수가 많을수록:
- ✅ 패시브 효과 ↑
- ✅ 카드 장착 슬롯 ↑
- ✅ 전멸 보험 ↑ (추가 목숨)
- ❌ 적 난이도 ↑ (칸 수 비례 스케일링)

### 2.2 칸 HP

- 각 칸은 고유 HP를 가짐
- **전투 중 적이 살아있는 동안 꼬리칸 HP 지속 감소**
- 적을 빨리 처리할수록 칸 데미지 적음 (DPS 체크)
- 스킬 수동 발동 → 빠른 처리 → 칸 보호 (수동 플레이의 의미)

```javascript
// 전투 중 매 초마다
function tickCarDamage(trainCars, aliveEnemies) {
  const tailCar = trainCars[trainCars.length - 1];
  const dps = aliveEnemies.reduce((sum, e) => sum + e.threatValue, 0);
  tailCar.hp -= dps;
  if (tailCar.hp <= 0) {
    destroyCar(tailCar);  // 폭발, 패시브 소멸
    // 다음 칸이 새 꼬리가 됨
  }
}
```

### 2.3 칸 파괴

- 칸 HP 0 → **폭발** (시각 효과)
- 해당 칸의 패시브 즉시 소멸
- 다음 칸이 새 꼬리칸이 됨
- 모든 칸 파괴 + 파티 전멸 → **런 실패**

### 2.4 파티 전멸

- 파티 전멸 시 → 꼬리칸 폭발 + 파티 부활 (HP 50%)
- 칸이 남아있는 한 계속 전투 가능
- 칸도 파티도 없으면 → 런 실패

## 3. 칸 종류

각 칸은 고유 패시브 + 연료(코스트) 생성량이 다름.

| 칸 | 패시브 | 연료/역 | HP |
|---|---|---|---|
| 탄약칸 | 파티 ATK +10% | 2 | 보통 |
| 의무칸 | 매 구간 끝 파티 HP 10% 회복 | 1 | 낮음 |
| 장갑칸 | 파티 DEF +5 | 1 | 높음 |
| 발전칸 | 연료 추가 생성 | 3 | 보통 |
| 화물칸 | 클리어 보상 +15% | 1 | 보통 |
| 수리칸 | 역 정차 시 다른 칸 HP 15% 회복 | 1 | 보통 |

(칸 종류/수치는 초안. 밸런스 조정 필요)

## 4. 카드 시스템 (덱빌딩)

### 4.1 카드 획득

- 구간(라운드) 클리어 시 **3택1** 카드 획득
- 카드 티어: 일반 / 희귀 / 전설 (구역 레벨에 따라 확률 변동)
- 획득한 카드는 손에 들고 있음 (아직 비활성)

### 4.2 역 정차 — 카드 활성화

- 2~3구간마다 역에 정차
- 보유 연료(칸들이 생성) 내에서 **카드 3~4장 장착/활성화**
- 각 카드에 활성화 코스트 있음
- 활성화된 카드 효과는 다음 역까지 (또는 런 종료까지) 지속

```
역 정차 시:
  보유 연료: 8 (탄약2 + 의무1 + 장갑1 + 발전3 + 화물1)
  손에 든 카드: 5장
  → 코스트 맞춰 3~4장 활성화
  → 남은 카드는 다음 역까지 보관
```

## 5. 구간 수 / 역 정차

| 구역 레벨 | 구간 수 | 역 정차 | 카드 획득 |
|---|---|---|---|
| Lv 1-3 | 4구간 | 2회 (2구간마다) | 3장 |
| Lv 4-6 | 5구간 | 2회 | 4장 |
| Lv 7-9 | 6구간 | 3회 (2구간마다) | 5장 |
| Lv 10 | 7구간 (보스) | 3회 | 6장 |

## 6. 연료 코스트 확장

Cargo 친화도 트리 노드로 연료 확장 가능:
- `정비공` (Lv4): 정차 수리량 +20%
- `보급관` (6B): 보급 크레이트 등장률 +50% → 추가 연료 드롭
- `화물 최적화` (Lv9): 연료 생성 +2/역

길드회관 G(화물 통제) 업그레이드:
- G3: 칸 HP +15%
- G6: 역 정차 시 카드 리롤 1회
- G9: 카드 장착 슬롯 +1
- G12: 전설 카드 등장 확률 2배

## 7. 서브 파견 vs 메인 전투

| | 메인 전투 | 서브 파견 |
|---|---|---|
| 참여 | 마을 전체 유휴 용병 | 지정 파티 2-3명 |
| 칸 편성 | 수동 | 없음 (고정 1칸) |
| 카드 | 3택1 획득 + 역 활성화 | 없음 |
| 스킬 | 수동 발동 | 없음 (일반공격만) |
| 칸 데미지 | 적 생존 시간 비례 | RunSimulator 시뮬 |
