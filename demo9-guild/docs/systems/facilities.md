# 시스템: 마을 시설

> 마을의 활동 거점. 길드 레벨에 따라 해금. **길드 회관(8 카테고리 메타 업그레이드)과는 별개**.
> 시설 = 수평 확장(새 활동 추가), 길드 회관 = 수직 강화(기존 활동 효율↑).

## 2. 시설별 상세

### 2.1 용병 모집소 ⚔

- 3명 표시, 마음에 들면 고용 (희귀도별 가격)
- 새로고침: 100G (게임 내 시간 10분 무료 갱신)
- 길드회관 B(인프라): 동시 표시 +N
- 평판 보너스: 35+ 시 epic 확률 +3%

### 2.2 보관함 📦

- 기본 캡: 길드 Lv × 10 (Lv1=10, Lv8=80)
- 비밀 금고 시 +50
- 길드회관 B 업그레이드로 +N
- 보안 컨테이너: 분실/자동판매 보호 슬롯

### 2.3 출발 게이트 🚪

- 메인 도전: 파티 편성 후 구역+레벨 선택
- 서브 파견: 빈 슬롯에 파티 보냄 (실시간 진행)
- 자동전투 잠금해제된 구역+레벨은 자동 모드 토글 표시

### 2.4 장비 제작소 🔨 (NPC 아이언)

- 제작: 소재 조합 → 신규 장비
- 강화: +1 ~ +6 (성공률 관리)
- 분해: 장비 → 소재
- 판매: 장비 → 골드 (70%)
- 동시 제작 슬롯: 1 → 4 (길드회관 B)

상세: [crafting.md](crafting.md)

### 2.5 경매장 🏛 (NPC 골든)

- 위탁: 본인 장비 등록 → 24시간 후 정산
- 입찰: 게시된 장비에 입찰 (다른 입찰자 시뮬)
- 호감도 진행으로 VIP 경매장 해금 (전설 등급 위주)
- 시세 변동: 마을 이벤트로 변동

### 2.6 훈련소 🏋

- 양성 특훈: 무작위 양성 특성 1개 부여
- 비용: 1,500G + 훈련 포인트 3
- 슬롯 제한: 최대 3양성/용병
- 길드회관 C 업그레이드로 비용 ↓

### 2.7 신전 ⛪ (NPC 사제)

- **정화 의식**: 음성 특성 1개 제거 (비용 + 쿨다운)
- **부활**: 부상 용병 회복 (사망 보호 모드에서)
- **명상**: 피로도 +50 즉시 (비용 50G)
- **축복**: 일시 버프 (런 1회)
- 사제 호감도 따라 비용/쿨다운 보정

### 2.8 정보소 🔍 (NPC 스피드카)

- 소문 구매 (등급별 가격)
- 의뢰 게시판 (평판 마일스톤 의뢰)
- 보스 정찰 (호감도 NPC 의뢰 라인)
- 길드회관 E 업그레이드로 갱신 가속

### 2.9 고급 모집소 👑

- 일반 모집소와 별개. 희귀+ 보장
- 가격: 일반 ×3, 무작위 양성 1개 보장
- 특성 재추첨: 800G로 음성 전체 재추첨

### 2.10 비밀 금고 🔒

- 보관함 +50 슬롯
- 보안 컨테이너: 자동판매/분실 보호 (자물쇠 슬롯)
- 길드 평판 60+ 보너스: 추가 +20 슬롯

## 3. 시설 해금 조건

```javascript
function canUnlockFacility(state, facilityKey) {
  const data = FACILITY_DATA[facilityKey];
  return state.guildLevel >= data.unlockLevel && state.gold >= data.cost;
}
```

해금은 1회성. 한 번 짓으면 영구. NG+ 진입 시 리셋 (메타 강화로 시작 해금 가능).

## 5. 시설 vs 길드 회관

| 구분 | 시설 | 길드 회관 |
|---|---|---|
| 해금 단위 | 시설 단위 (이진) | 단계 (8 카테고리 × 12) |
| 비용 | 골드 단발 | 골드 누적 (1.5^단계) |
| 효과 | 새 활동 추가 | 기존 활동 효율 ↑ |
| 예시 | 신전 짓기 (정화 가능해짐) | 회복 C9 (정화 비용 -30%) |

## 6. 길드회관 연동

- 길드회관 D(자동화): 시설 일부 자동 모드 해금
- 길드회관 E(정보): 정보소 시세 가속
- 길드회관 C(휴식): 신전/술집 효율 ↑
- 길드회관 B(인프라): 보관함/제작소 슬롯 ↑

## 8. gameState 연동

```javascript
unlockedFacilities: [],   // 해금된 시설 키 배열
guildLevel: 1,
guildXp: 0,
storage: [],              // 보관함 아이템
secureContainer: [],      // 보안 컨테이너 아이템 (비밀금고)
roster: [],               // 보유 용병
recruitPool: [],          // 모집소 현재 풀
trainingPoints: 0,
training: {}              // 진행 중인 훈련
```

기존 GuildManager.js의 createDefaultState() 그대로 사용. game-state.md의 신규 필드와 별도.
