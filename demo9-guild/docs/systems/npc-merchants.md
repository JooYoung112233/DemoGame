# 시스템: NPC 상인 (타르코프식)

> 8명의 NPC. 호감도 4티어 + 호감도 의뢰 라인 + 물물교환 + 캐릭터성.
> 단순 거래소가 아닌 **각자 시스템 하나**로 동작.

## 2. 호감도 시스템

### 2.1 4티어 (확장)

| 티어 | 호감도 | 이름 | 해금 |
|---|---|---|---|
| 0 | 0-9 | 낯선 | 기본 매물만 |
| 1 | 10-29 | 친근한 | 가격 -5%, uncommon 매물, 의뢰 라인 1단계 |
| 2 | 30-59 | 신뢰하는 | -10%, rare 매물, 의뢰 2단계, **물물교환 1슬롯** |
| 3 | 60-99 | 동맹 | -15%, epic 매물, 의뢰 3단계, **물물교환 2슬롯**, 특수 서비스 |
| 4 | 100 | 절친 | -20%, legendary 매물 (확률), 의뢰 4단계 (메인 스토리), **물물교환 3슬롯**, NPC 고유 보상 |

### 2.2 호감도 누적

- NPC와 거래 (1회): **+0.5**
- NPC 의뢰 완료: **+8** (단계별)
- NPC 선호 자원 선물: +1~3 (NPC별 다름)
- 마을 이벤트 NPC 호의 선택: +2~5
- 평판 마일스톤 도달 시 모든 NPC +5

### 2.3 호감도 패널티

- NPC 의뢰 실패: -3
- 마을 이벤트에서 NPC 적대 선택: -5~10
- 음성 특성 `arrogant` 보유자가 거래: 추가 -0.1

## 4. 물물교환 시스템

호감도 2티어부터 NPC별 물물교환 슬롯 등장. 골드로는 살 수 없는 매물.

### 4.1 물물교환 구조

```
NPC 매물 옆에 [물물교환] 슬롯:
  요구: 가죽 5개 + 혈정석 1개
  → 미라의 비전 포션 (영구 ATK +3 1명)
```

### 4.2 예시 물물교환

| NPC | 요구 | 결과 |
|---|---|---|
| 크로그 | 핏로드의 송곳니 1 + 강철 5 | 핏로드의 검 (epic) |
| 미라 | 저주 유물 3 + 마력의 가루 5 | 영구 음성 면역 부적 |
| 그림자 | 평판 -10 자발적 | 저주 전설 장비 |
| 아이언 | 분해 토큰 50개 | 자동 분해 기능 영구 |
| 골든 | 전설 장비 위탁 1 | 무작위 epic 장비 |

## 5. 매물 로테이션

각 NPC 개별 인벤토리. 게임 내 시간 1시간마다 갱신.

- 호감도 ↑ = 슬롯 수 ↑ (3 → 5 → 7 → 8 → 10)
- 호감도 ↑ = 등급 ↑
- 길드회관 E3: 갱신 주기 -30%
- 평판 마일스톤: 모든 NPC 슬롯에 보너스 등급 1개

## 6. NPC 캐릭터성

### 대화 시스템 (간소화)

- NPC 호버/접근 시 짧은 대사 (호감도/평판/시간대에 따라 다름)
- 의뢰 등장 시 NPC 일러스트 + 본문 (마을 이벤트와 비슷한 형식)
- 호감도 티어 도달 시 특별 대사 (티어업 이벤트)

### NPC별 성격

| NPC | 성격 | 선호 자원 | 싫어하는 행동 |
|---|---|---|---|
| 크로그 | 호탕, 직설 | 강철, 무기 소재 | 약속 어김 |
| 미라 | 신비, 차분 | 약초, 마력의 가루 | 음성 방치 |
| 헤르타 | 깐깐, 완벽주의 | 갑옷 소재 | 갑옷 분해 |
| 골든 | 사교적, 약삭빠름 | 희귀 장비 | 위탁 회수 |
| 스피드카 | 의심 많음, 신중 | 정보, 소문 | 정보 누설 |
| 아이언 | 묵묵, 장인 | 강화 재료 | 강화 실패 방치 |
| 그림자 | 위험, 카리스마 | 저주 자원, 음성 | 평판 추구 |
| 사제 | 자비, 엄격 | 평판, 양성 | 저주 추구 |

→ 어떤 NPC를 키울지 = 어떤 길드를 만들지 (개성)

## 7. NPC 간 관계

일부 NPC는 서로 영향:
- 그림자 ↔ 사제: 한쪽 호감 +1 시 다른 쪽 -0.5 (대립)
- 크로그 ↔ 헤르타: 동시 진행 시 보너스 (협력)
- 골든 ↔ 미라: 경매 시 미라 약초 매물 등장 가속
