# 시스템: 소문 (Rumors)

> 마을에서 얻는 정보 단서. 후속 이벤트/보스 약점/숨겨진 콘텐츠 해금에 사용.

## 1. 컨셉

- NPC 스피드카(정보상) 중심으로 거래
- 마을 이벤트 부산물로도 입수
- 단순 텍스트가 아닌 **사용 가능한 자원** (소비 시 효과 발동)
- 시간 지나면 만료 (구식 정보)

## 2. 소문 카테고리

| 카테고리 | 효과 |
|---|---|
| 🗺 **구역 정보** | 다음 메인 전투에서 적 구성 미리 공개 |
| 👹 **보스 단서** | 보스 약점/패턴 단서 (보스 피해 +10%) |
| 💰 **상인 정보** | 다음 NPC 매물 로테이션 미리보기 |
| ❓ **숨겨진 이벤트** | 특정 마을 이벤트 강제 트리거 |
| 🔮 **유적 단서** | 비밀방/저주 의식방 등장 확률↑ |
| ⚔ **음모** | 평판 큰 변화 이벤트 트리거 |

## 3. 입수처

| 경로 | 비중 |
|---|---|
| NPC 스피드카 구매 | 50% — 골드 지불 |
| 마을 이벤트 보상 | 20% — 무작위 |
| 술집/신전 행동 보상 | 15% — 호감도 연동 |
| 던전 이벤트 방 (BO) | 10% — 탐색 보상 |
| 친화도 트리 노드 | 5% — 특정 노드 보유 |

## 4. 등급

| 등급 | 효과 강도 | 만료 시간 |
|---|---|---|
| 일반 | 단발성 효과 | 30분 (게임 내 시간) |
| 고급 | 중간 효과 | 1시간 |
| 비밀 | 큰 효과 | 2시간 |

## 5. 사용

- 인벤토리에 보관 (최대 10개, 길드회관 E로 확장)
- 적절한 타이밍에 사용:
  - 구역 정보 → 메인 전투 입장 직전
  - 보스 단서 → 보스전 직전
  - 숨겨진 이벤트 → 마을에서 사용
- 일부는 자동 발동 (보유만 해도 효과)

## 6. 예시 소문

| 이름 | 등급 | 효과 |
|---|---|---|
| BP Lv5 적 명단 | 일반 | 다음 BP Lv5 도전 시 적 구성 공개 |
| 핏로드의 약점 | 고급 | 다음 BP 보스전에서 보스 피해 +15% |
| 무기상의 비밀 재고 | 일반 | 다음 크로그 매물에 희귀+ 보장 |
| 잊혀진 신전 위치 | 비밀 | 마을 이벤트 "잊혀진 신전" 강제 발동 |
| 그림자 군주의 진명 | 비밀 | BO 보스전 1페이즈 즉시 종료 |
| 도적단의 음모 | 고급 | 평판 변동 이벤트 트리거 (선택) |

## 7. 길드회관 E(정보) 연동

| 단계 | 효과 |
|---|---|
| E3 | 소문 보관 한도 +3 (총 13) |
| E6 | 소문 만료 시간 +50% |
| E9 | 마을 이벤트 보상에서 소문 확률 ×2 |
| E12 | 모든 소문 자동 공개 (구매 불요, 무한 보관) |

## 8. UI

- 인벤토리 탭 (별도): 소문 목록 + 만료 카운트다운
- 사용 가능한 곳에서 호버 시 소문 적용 미리보기
- 만료 임박 시 알림

## 10. 시스템 연동

- **NPC 스피드카**: 호감도 높을수록 좋은 등급 판매
- **마을 이벤트**: 일부 이벤트 선택지 보상으로 소문 드롭
- **메인 전투**: 입장 직전 슬롯에서 소문 사용 가능
- **보스전**: 특정 소문 보유 시 자동 약점 노출
- **친화도 트리**: BO `occultist` (Lv5) 보유 시 비밀 소문 +1 슬롯
