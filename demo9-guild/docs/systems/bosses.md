# 시스템: 구역 보스 (페이즈 패턴)

> 3구역 보스 각각 3페이즈. 보스 처치 시 다음 구역 해금 + 전설 장비 + 보스 토큰.

## 1. 공통 룰

- **등장**: 각 구역 Lv10 메인 전투의 3라운드 (BP) / 7구간 (Cargo) / 보스방 (BO)
- **페이즈**: HP 비율로 전환 (Phase 1 100-66%, Phase 2 66-33%, Phase 3 33-0%)
- **페이즈 전환**: 보스가 잠시 무적, 패턴 변경, 화면 효과
- **전설 드롭 보장**: 첫 처치 시 100%, 이후 30%
- **보스 토큰 드롭**: 첫 처치 시 1개 (친화도 캡스톤 해금용)

## 2. 핏로드 (Blood Pit 보스)

### 2.1 컨셉
혈투장의 지배자. 거대한 검투사 + 광폭한 야수의 결합.

### 2.2 페이즈

**Phase 1 (100-66%) — 검투 단계**
- 일반 근접 공격 (ATK 보통, 빠른 공속)
- 5초마다 회전 베기 (반경 100px AoE)
- 핏 게이지 충전 속도 +20% (플레이어 보너스)

**Phase 2 (66-33%) — 광폭화 단계**
- 공속 +50%, 이동속도 +30%
- 8초마다 돌진 (직선, 경로상 모든 아군 ATK ×2 피해)
- "관중의 야유" — 보스 처치 미스 시 핏로드 ATK +10% 누적

**Phase 3 (33-0%) — 핏로드의 분노**
- 모든 스탯 +30%
- 매 5초마다 핏 폭발 (전체 화면 AoE, 회피 가능 영역 표시)
- 한쪽 팔이 파괴되어 공격 패턴 변경 (반대 손 휘두르기)
- 처치 시 폭발 (남은 적 전멸 + 보너스 보상)

### 2.3 보상

| 보상 | 확률 |
|---|---|
| 핏로드의 송곳니 (전설 소재) | 100% (첫 처치) |
| `bp_token` 보스 토큰 | 1개 (첫 처치) |
| 전설 무기 (BP 테마) | 60% / 재처치 30% |
| 평판 +5 | 확정 |

## 3. 화물 제독 (Cargo 보스)

### 3.1 컨셉
선장 + 거대 기계 골렘. 화물열차 최종역의 수호자.

### 3.2 페이즈

**Phase 1 (100-66%) — 선장의 명령**
- 호위병 소환 (3마리 일반 몹, 3페이즈 종료까지 계속 리스폰)
- 6초마다 함포 사격 (직선 관통, 회피 가능)
- 본체는 후방에서 명령만 내림 (이동 느림)

**Phase 2 (66-33%) — 골렘 전환**
- 호위병 전멸 시 골렘 모드로 변환 (선장이 골렘 안으로)
- 강력한 근접 공격 (ATK ×1.5, 느린 공속)
- 12초마다 증기 분출 (4방향 직선)
- 칸이 데미지 받음 (CARGO 칸 시스템과 연동)

**Phase 3 (33-0%) — 최종 화물**
- 골렘 폭주, 이동속도 +50%
- 매 8초마다 화물 투척 (랜덤 위치 AoE)
- 모든 카드 효과 -20% (저주성)
- 처치 시 골렘 폭발 (대형 보상 박스 드롭)

### 3.3 보상

| 보상 | 확률 |
|---|---|
| 화물 제독의 인장 (전설 소재) | 100% (첫 처치) |
| `cargo_token` 보스 토큰 | 1개 (첫 처치) |
| 전설 갑옷 (Cargo 테마) | 60% / 재처치 30% |
| 평판 +5 | 확정 |

## 4. 그림자 군주 (Blackout 보스)

### 4.1 컨셉
저택을 저주한 고대 영체. 무형 + 다중 그림자.

### 4.2 페이즈

**Phase 1 (100-66%) — 그림자 단계**
- 본체는 무적, 3개의 그림자 분신만 공격 가능
- 분신은 일반 몹 수준 (HP 낮음, 빠름)
- 분신 모두 처치 시 본체 잠시 노출 (5초간 피격 가능)
- 본체에 데미지 누적 시 페이즈 종료

**Phase 2 (66-33%) — 저주의 강림**
- 본체 무적 해제, 직접 공격 가능
- 모든 아군에 저주 디버프 (-10% 모든 스탯)
- 매 7초마다 어둠 영역 생성 (이동 차단)
- "저주의 화신" — 저주 레벨 비례 추가 피해

**Phase 3 (33-0%) — 그림자 군주**
- 본체가 분열 (3개 본체, 각각 33% HP)
- 각 본체는 다른 패턴 (근접/원거리/소환)
- 모두 처치해야 클리어
- "심연의 외침" — 매 10초 전체 화면 정신 디버프

### 4.3 보상

| 보상 | 확률 |
|---|---|
| 그림자 군주의 가면 (전설 소재) | 100% (첫 처치) |
| `bo_token` 보스 토큰 | 1개 (첫 처치) |
| 전설 장식구 (BO 테마) | 60% / 재처치 30% |
| 평판 +5 | 확정 |
| 추가: 저주 레벨 5+ 도전 시 → 추가 전설 1개 보장 | |

## 5. 보스 강화 (반복 처치)

- 첫 처치 후에도 메인 전투로 재도전 가능 (보스 토큰은 추가 안 나옴)
- 매 처치마다 보스 강화 (+10% 스탯) → 도전성 유지
- 강화 보스 처치 시 추가 보상 (소재 보장, 골드↑)
- 10회 처치 = 보스 마스터 칭호 + 영구 버프

## 6. 페이즈 전환 UI

- 페이즈 전환 시 화면 흔들림 + 색상 변화
- 보스 HP바에 페이즈 구간 표시 (3구역으로 나뉜 바)
- 페이즈 변경 시 텍스트 알림 (예: "광폭화!")
- 페이즈별 BGM 변경

## 7. 친화도 / 길드회관 연동

- 친화도 캡스톤 보스 한정 보너스 (예: BP `혈의 군주` = ATK +20%)
- 길드회관 F/G/H 단계 12: 해당 구역 보스 약점 노출
- 친화도 트리 `~ 사냥꾼` 노드: 보스 데미지 +15%

## 8. 보스전 진입 게이트

- BP 보스: 구역 레벨 10 도달 후 (총 9 클리어)
- Cargo 보스: BP 보스 처치 후 + Cargo Lv10
- BO 보스: Cargo 보스 처치 후 + BO Lv10
- 보스전은 메인 파티 전용 (서브 파견 불가)
