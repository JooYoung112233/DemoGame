# 시스템: 제작 / 강화 / 분해

> NPC 아이언(대장장이)에서 운영. 구역별 소재 + 타르코프식 레시피 + 제작 전용 아이템.

## 1. 4가지 기능

| 기능 | 설명 | 비용 |
|---|---|---|
| **제작** | 소재 조합으로 새 장비 생성 | 소재 + 골드 |
| **강화** | 기존 장비 +1/+2/... | 같은 등급 소재 + 골드 |
| **분해** | 장비를 소재로 변환 | 무료 (장비 소모) |
| **판매** | 장비를 골드로 변환 | 정가 70% |

## 2. 구역별 소재

각 구역에서 고유 소재 드롭. 타르코프처럼 **소재 자체가 가치**, 일부는 중간재.

### 2.1 Blood Pit 소재

| 등급 | 이름 | 드롭처 | 용도 |
|---|---|---|---|
| 일반 | 가죽 | BP 일반 몹 | 가죽 갑옷 제작 |
| 일반 | 뼛조각 | BP 일반 몹 | 무기 강화 |
| 희귀 | 혈정석 | BP 엘리트 | 출혈 무기 제작 |
| 전설 | 핏로드의 송곳니 | BP 보스 | 전설 장비 제작 |

### 2.2 Cargo 소재

| 등급 | 이름 | 드롭처 | 용도 |
|---|---|---|---|
| 일반 | 강철 조각 | Cargo 일반 몹 | 갑옷 제작 |
| 일반 | 기계 부품 | Cargo 일반 몹 | 장비 강화 |
| 희귀 | 마법 동력석 | Cargo 엘리트 | 증기 장비 제작 |
| 전설 | 화물 제독의 인장 | Cargo 보스 | 전설 장비 제작 |

### 2.3 Blackout 소재

| 등급 | 이름 | 드롭처 | 용도 |
|---|---|---|---|
| 일반 | 어둠의 정수 | BO 일반 몹 | 저주 장비 강화 |
| 일반 | 영혼 조각 | BO 일반 몹 | 장비 강화 |
| 희귀 | 저주 유물 | BO 엘리트/저주방 | 저주 장비 제작 |
| 전설 | 그림자 군주의 가면 | BO 보스 | 전설 장비 제작 |

### 2.4 공통 소재 (도시/이벤트)

| 등급 | 이름 | 획득처 | 용도 |
|---|---|---|---|
| 일반 | 기본 금속 | 마을 이벤트, 분해 | 모든 제작 기본 |
| 희귀 | 마력의 가루 | NPC 미라(약초상) | 마법 강화 |
| 전설 | 순수 결정 | 마을 이벤트 (드물게) | 전설 제작 |

## 3. 레시피 (초안)

### 3.1 일반 제작 (드롭 가능 장비)

| 결과물 | 소재 | 골드 |
|---|---|---|
| 강철검 (uncommon) | 강철 조각 ×3, 가죽 ×2 | 200 |
| 강철 갑옷 (uncommon) | 강철 조각 ×4, 가죽 ×3 | 250 |
| 마력 지팡이 (rare) | 마법 동력석 ×2, 마력의 가루 ×3 | 600 |
| 가죽 활 (uncommon) | 가죽 ×4, 뼛조각 ×2 | 220 |

### 3.2 구역 특수 장비 제작

| 결과물 | 소재 | 골드 |
|---|---|---|
| 피에 물든 철검 (rare) | 강철 조각 ×3, 혈정석 ×2 | 500 |
| 증기 갑옷 (rare) | 강철 조각 ×4, 마법 동력석 ×2, 기계 부품 ×3 | 700 |
| 저주받은 반지 (rare) | 영혼 조각 ×3, 저주 유물 ×1 | 600 |

### 3.3 제작 전용 (드롭 불가능)

> 이 장비들은 **제작으로만** 획득 가능. 강력한 효과, 3구역 소재 전부 필요.

| 결과물 | 등급 | 소재 | 골드 |
|---|---|---|---|
| 삼위일체 검 | epic | 혈정석 ×2 + 마법 동력석 ×2 + 저주 유물 ×2 | 2000 |
| 신화의 갑옷 | epic | 핏로드의 송곳니 ×1 + 화물 제독의 인장 ×1 | 5000 |
| 군주의 왕관 | legendary | 보스 소재 3종 각 1 + 순수 결정 ×3 | 10000 |

→ **모든 구역 보스를 잡아야만** 제작 가능 → 진행 동기 부여

## 4. 강화 시스템

| 강화 단계 | 효과 | 소재 | 골드 | 성공률 |
|---|---|---|---|---|
| +1 → +2 | 스탯 +5% | 같은 등급 소재 ×1 | 100 | 100% |
| +2 → +3 | 스탯 +10% | 같은 등급 소재 ×2 | 300 | 90% |
| +3 → +4 | 스탯 +15% | 같은 등급 소재 ×3 | 800 | 70% |
| +4 → +5 | 스탯 +25% | 희귀 소재 ×1 | 2000 | 50% |
| +5 → +6 | 스탯 +35% | 희귀 소재 ×2 | 5000 | 30% |

- 실패 시: +1 단계 하락 (보호 부적 사용 시 유지)
- 보호 부적: NPC 아이언 판매, 1회용 1000G
- 길드회관 D(자동화) 또는 친화도 트리로 성공률 보정 가능

## 5. 분해 시스템

장비를 소재로 변환. 분해 결과는 장비 종류/등급/구역에 따라 결정.

| 장비 등급 | 평균 소재 회수 |
|---|---|
| common | 일반 소재 1-2개 |
| uncommon | 일반 소재 2-3개 (가끔 희귀 1) |
| rare | 일반 2-4개 + 희귀 1 |
| epic | 희귀 2-3개 + 가끔 전설 1 |
| legendary | 전설 소재 1-2개 |

- 구역 특수 장비 분해 시 해당 구역 소재 보장
- 일괄 분해 가능 (자동화 모드)

## 6. 제작 시간 / 슬롯

타르코프처럼 실시간 제작:

| 결과물 등급 | 제작 시간 |
|---|---|
| uncommon | 30초 |
| rare | 2분 |
| epic | 10분 |
| legendary | 30분 |

- 동시 제작 슬롯: 기본 1개, 길드회관 B(인프라) 확장 시 +1씩 (최대 4)
- 제작 중 다른 활동 가능 (서브 파견, 메인 전투 등)
- 완료 시 알림

## 7. NPC 아이언 호감도 연동

| 아이언 호감도 | 효과 |
|---|---|
| 낯선 (0) | 기본 |
| 친근한 (5) | 강화 비용 -5% |
| 신뢰하는 (15) | 제작 시간 -15%, 강화 성공률 +5% |
| 동맹 (30) | 강화 비용 -15%, 전설 제작 해금 |
| 절친 (60) | 제작 시간 -30%, 보호 부적 50% 할인 |

(NPC 호감도 상세는 [npc-merchants.md](npc-merchants.md))

## 9. 게이트 / 진행 동기

- 일반 제작: 길드 Lv2부터 (NPC 아이언 해금)
- 구역 특수 장비: 해당 구역 첫 클리어 후
- 제작 전용 epic: 모든 구역 첫 클리어 후
- 제작 전용 legendary: 모든 구역 보스 처치 후
