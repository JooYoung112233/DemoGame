## 3. 전설 특성 (8개 목표)

> legendary 등급 용병만 보유 가능. 80레벨 친화도 또는 보스 토큰으로 부여.

| 키 | 이름 | 효과 |
|---|---|---|
| `champion` | 챔피언 | 모든 스탯 +10%, Bond 누적 +50% |
| `harbinger` | 전조 | 보스 등장 시 파티 전체 ATK +20% (1라운드) |
| `phoenix` | 불사조 | 사망 시 1회 부활 (HP 50%), 런당 1회 |
| `ironbound` | 강철 결속 | 같은 파티 모든 용병 DEF +3 |
| `bloodseeker` | 피의 추적자 | 킬 시 추가 골드 +25% (개인) |
| `shadowmaster` | 그림자 대가 | BO 저주 효과 절반 |
| `crowned` | 왕관 보유자 | 길드 평판 획득 시 +5 추가 |
| `eternal` | 영원의 의지 | 모든 디버프 지속 시간 -50% |

## 4. 슬롯 제한

- 양성: 최대 3슬롯
- 음성: 최대 2슬롯
- 전설: 1슬롯 (legendary 등급만)
- 모집 시 희귀도에 따라 초기 특성 수 결정 (RARITY_DATA 참고)

## 5. 부여 / 변경

상세 경로는 [traits.md](traits.md) 참고.

- 양성 추가: 훈련소 특훈, 마을 이벤트 보상
- 음성 추가: 마을 이벤트 결과 (가족 위기 등), 음성 발현
- 음성 제거: 신전 정화, 모집소 재추첨, BO 정화의 샘
- 전설 부여: 친화도 80 도달 / 보스 토큰 / 전설 의뢰 보상

## 6. UI 표시

### 용병 상세 화면
```
[🛡] 발렌타인 (희귀, 전사 Lv12)
양성: 강철 팔, BP 베테랑, 신속
음성: 성마름
전설: -

영구 효과:
+ ATK +23%, DEF +5, SPD +10%
+ BP에서 ATK +15% (구역 한정)
- Bond 누적 50% 확률 -50%
```
