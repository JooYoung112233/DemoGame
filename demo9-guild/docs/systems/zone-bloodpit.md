# 구역 전투: Blood Pit (혈투장)

> BP 전용 전투 메카닉. 다른 구역은 각각 별도 문서.

## 1. 기본 구조

```
입장 → 라운드1(자동전투) → 클리어 → [3택1 선택] → 라운드2 → 클리어 → [3택1 선택] → 라운드3 → 클리어 → 보상 정산
```

- **라운드 수**: 3 (고정)
- **선택 기회**: 2회 (라운드1→2, 라운드2→3 사이)
- **선택지**: 3택1, 런 한정 임시 능력 (런 끝나면 초기화)
- **전투 방식**: 일반공격 자동 + **스킬은 플레이어 직접 발동**

## 2. 전투 개입 (공통 원칙)

모든 구역 메인 전투의 핵심 개입 요소:

- **일반공격**: 자동 (오토배틀러)
- **스킬**: 플레이어가 타이밍/대상 맞춰 직접 발동 (탭/클릭)
- **서브 파견과의 차이**: 서브는 일반공격만 → 스킬 없이는 상위 레벨 클리어 불가
- 스킬 타이밍이 실력 = 직접 플레이의 의미

### 자동전투 잠금해제

- **첫 클리어**: 반드시 수동 (스킬 직접 발동)
- **클리어 후**: 해당 구역+레벨의 **자동전투 해금** — 스킬도 AI가 자동 발동
- 자동 AI는 플레이어보다 스킬 타이밍이 비최적 → 수동보다 약간 약함
- 이미 깬 레벨 반복 파밍 시 편의 기능
- 길드회관 D(자동화) 업그레이드로 자동 AI 정확도 향상 가능

## 3. 핏 게이지

**역할**: 클리어 보상 배율기.

| 항목 | 값 |
|---|---|
| 범위 | 0% ~ 100% |
| 충전 | 라운드 클리어 시 고정 +33% (3라운드 전클 = 100%) |
| 보상 배율 | 게이지% × 1.0 → 최대 **2.0배** (0%=1.0x, 100%=2.0x) |
| 적용 대상 | 골드, XP, 장비 드롭, 친화도 XP 전부 |

```javascript
function getPitRewardMultiplier(pitGauge) {
  return 1.0 + (pitGauge / 100);  // 0%=1.0x, 50%=1.5x, 100%=2.0x
}
```

라운드 실패(전멸) 시 해당 라운드 게이지 충전 없음 → 보상 배율 하락.

## 4. 라운드별 적 구성

| 라운드 | 적 구성 | 난이도 |
|---|---|---|
| 1 | 일반 몹 웨이브 | 구역 레벨 기본 |
| 2 | 강화 몹 + 엘리트 1체 | 기본 × 1.3 |
| 3 | 엘리트 다수 또는 보스 | 기본 × 1.6 |

구역 레벨(1-10)에 따라 적 스탯 스케일링. Lv10 3라운드 = 보스전.

## 5. 선택지 시스템 (뱀서식)

### 4.1 등장 규칙

```javascript
function generateBPChoices(zoneLevel) {
  // 티어 확률: 구역 레벨에 따라 희귀/전설 확률 증가
  const tierChance = {
    common:    Math.max(0.4, 0.7 - zoneLevel * 0.03),
    rare:      Math.min(0.4, 0.25 + zoneLevel * 0.02),
    legendary: Math.min(0.2, 0.05 + zoneLevel * 0.01)
  };
  // 3개 뽑기, 중복 없음
  return pickWeightedUnique(BP_CHOICE_POOL, tierChance, 3);
}
```

## 6. 길드회관 F(핏 통제) 연동

길드회관 F 카테고리 업그레이드가 BP 전투를 변경:
- F3: 핏 게이지 감소 속도 -20%
- F6: 선택지 리롤 1회 가능
- F9: 선택지 4택1로 확장
- F12: 전설 등장 확률 2배

(상세는 [guild-hall.md](guild-hall.md) F 참고)
