# 구역 전투: Blood Pit (혈투장)

> BP 전용 전투 메카닉. 다른 구역은 각각 별도 문서.
> **v4 개정 (2026-05)**: 카드(뱀서식 3택1) 제거 → 관중 난입 + 쉬는 곳 시스템 도입.

## 1. 기본 구조 (v4)

```
[수동 모드 — 다키스트 베이스 + 혈투장 향]
입장 → R1 → (확률) 쉬는 곳? → R2 → (확률) 쉬는 곳? → R3 → 보상 정산
        ↑                       ↑
        라운드 중: 핏 게이지 임계점에 "관중 난입"

[자동 모드 — 관전/서브 파견]
입장 → 백그라운드 시간 경과 → 마을 진입 시 결과 수령
(시각적 관전은 옵션, 쉬는 곳 자동 처리)
```

- **라운드**: 3 (고정). R1·R2 일반, R3 보스/엘리트
- **카드 시스템**: ❌ 제거 (구. 뱀서식 3택1)
- **신규 1**: 라운드 중 핏 게이지 임계점에 **관중 난입** (적 추가 투입)
- **신규 2**: 라운드 사이 확률로 **쉬는 곳(대기실)** 등장 — 길드회관 F 해금 필요
- **전투 방식**: 다키스트 던전 스타일 (4인 일렬 + SPD 턴제) — 상세: [manual-battle-darkest.md](manual-battle-darkest.md)

## 2. 전투 개입 (공통 원칙)

- **일반공격**: 자동
- **스킬**: 플레이어가 타이밍/대상 직접 선택
- **서브 파견과의 차이**: 서브는 일반공격만, 스킬 없이는 상위 레벨 클리어 불가

### 자동전투 잠금해제

- 첫 클리어는 수동 필수
- 클리어 후 해당 구역+레벨 자동전투 해금 (AI가 스킬도 발동, 수동보다 약함)
- 길드회관 D 업그레이드로 자동 AI 정확도 향상 가능

## 3. 핏 게이지 (v4 개정)

**역할**: 클리어 보상 배율기 + **관중 난입 트리거**.

### 3.1 충전 방식 — 라운드 중에도 차오름

| 행동/이벤트 | 게이지 |
|---|---|
| 크리티컬 적중 | +3% |
| 출혈 효과 부여(첫 부여 시) | +1% |
| 화상 효과 부여(첫 부여 시) | +1% |
| 둔화·도발 등 그 외 디버프 부여 | +0.5% |
| 스킬 발동 | +5% |
| 적 처치 — 일반 | +5% |
| 적 처치 — 엘리트 | +10% |
| 적 처치 — 보스 | +20% |
| 라운드 클리어 | +10% (보너스) |

> 설계 의도: "관중이 좋아하는 것"(=피의 향연 — 크리·DoT·스킬·종결) 중심으로 차오름.

### 3.2 범위 / 배율

| 항목 | 값 |
|---|---|
| 범위 | 0% ~ 100% |
| 보상 배율 | 게이지% × 1.0 → 최대 **2.0배** (0%=1.0x, 100%=2.0x) |
| 적용 대상 | 골드, XP, 장비 드롭, 친화도 XP |

```javascript
function getPitRewardMultiplier(pitGauge) {
  return 1.0 + (pitGauge / 100);  // 0%=1.0x, 100%=2.0x
}
```

라운드 실패(전멸) 시 게이지 그대로 유지하되 보상 없음.

### 3.3 임계점 (관중 난입 트리거)

| 임계점 | 효과 |
|---|---|
| 33% 도달 | 1차 관중 난입 (1회) |
| 66% 도달 | 2차 관중 난입 (1회) |

같은 라운드 내에서 임계점은 1회만 발동. 다음 라운드에서 임계점 재무장(이미 넘은 임계점은 다시 안 터짐 — 게이지가 같은 라운드에서 떨어지진 않으므로 자연스러움).

## 4. 라운드별 적 구성

| 라운드 | 적 구성 | 난이도 |
|---|---|---|
| 1 | 일반 몹 웨이브 | 구역 레벨 기본 |
| 2 | 강화 몹 + 엘리트 1체 | 기본 × 1.3 |
| 3 | 엘리트 다수 또는 보스 | 기본 × 1.6 |

구역 레벨(1-10)에 따라 적 스탯 스케일링. Lv10 R3 = 보스전.

## 5. 관중 난입 시스템 (v4 신규)

> 카드(3택1) 자리를 대체하는 BP의 핵심 차별점.

### 5.1 발동 조건

- **언제**: 라운드 중 핏 게이지가 33%/66% 임계점을 넘는 순간
- **어디**: R1·R2 기본 적용, **R3 보스전은 기본 제외**(아래 5.4 예외 참고)
- **횟수**: 같은 라운드 내 임계점당 1회 (최대 2회/라운드)

### 5.2 난입 종류 — 두 가지

**시작 시점에 95% / 5% 확률 추첨**:

#### A. 도전적 난입 (95%)

라운드 적의 ~50~110% 강도로 새 적 등장. 처치 시 핏 게이지 보너스, 안 잡으면 패널티.

| 구역 레벨 | 33% 임계점 난입 | 66% 임계점 난입 |
|---|---|---|
| Lv1-3 | 잡졸 1명 (~70%) | 잡졸 2명 (~50%) |
| Lv4-6 | 잡졸 2명 (~50%) | 잡졸 1명 + 자객 1명 (~70%) |
| Lv7-9 | 자객 1명 (~80%) | 자객 1명 + 도전자 1명 (엘리트) |
| Lv10 | 자객 2명 (~80%) | 도전자 1명 (임팩트, ~110%) |

#### B. 도파밍 난입 (5%) — 이벤트성

"관중이 환호하며 약한 도전자를 던져넣음" 연출. 일반 적의 30~50% 강도. 처치 시 아이템 와르르(골드 ×3, 장비 드롭률 +200%).

> 한 라운드에서 양쪽 임계점 모두 도파밍 난입이 뜨면 보상 합산 — 럭키 데이.

### 5.3 처치/방치 결과

| 결과 | 핏 게이지 변동 |
|---|---|
| 난입 적 처치 (라운드 내) | **+5%/마리** |
| 난입 적 미처치 (라운드 완주) | **-10%/마리** |

- **라운드 클리어 조건**: 원래 적만 처치하면 종료. 난입 적은 무시하고 끝낼 수 있음(전략적 선택).
- 다만 미처치 패널티가 처치 보너스의 2배이므로 잡는 게 기본 이득.
- AoE·다중 타깃 클래스 가치 ↑.

### 5.4 보스전(R3) 간헐 등장

R3은 기본 제외이지만 **드물게** 보스전에서도 관중 난입 발생:

| 조건 | 확률 |
|---|---|
| 보스 HP 50% 이하 진입 시 | 15% |
| 길드회관 F 후반(F9+) 해금 시 | 위 확률 ×2 |

보스전 난입은 무조건 도전적 난입(자객 1체 ~ 도전자 1체).

## 6. 쉬는 곳 (대기실 — v4 신규)

> 카드(3택1) 자리를 보완하는 회복·재정비 페이지. 길드회관 F 해금 전에는 없음.

### 6.1 등장 조건

- **선결**: 길드회관 F의 해당 단계 해금 (§7 참고)
- **위치**: 라운드 클리어 후, 다음 라운드 시작 전
- **카운트**: 쉬는 곳은 라운드 카운트에 포함 X (즉 "R1 → 쉬는 곳 → R2"로 R2가 정상 진행)
- **횟수**: 한 런(BP 입장 1회)에서 최대 2회까지 등장 가능

### 6.2 등장 확률 공식

```javascript
function shouldShowRestRoom(zoneLevel, currentChance) {
  // 등장 시 0%로 리셋 → 다음 라운드부터 증가율 적용
  const baseRate    = getBaseRate(zoneLevel);     // 쉬운 5-10% / 어려운 5%
  const increaseRate = getIncreaseRate(zoneLevel); // 라운드당 +3~5%p
  const cap = 0.5;                                  // 최대 50%

  // currentChance 가 baseRate 이상이고 cap 도달 전이면 증가
  return Math.random() < Math.min(cap, currentChance);
}

function getBaseRate(zoneLevel) {
  // 쉬운 구역(Lv1-4)은 초기값 높음, 어려운 구역(Lv7+)은 낮음
  if (zoneLevel <= 4) return 0.10;
  if (zoneLevel <= 7) return 0.07;
  return 0.05;
}

function getIncreaseRate(zoneLevel) {
  if (zoneLevel <= 4) return 0.05;  // 5%p/라운드
  if (zoneLevel <= 7) return 0.04;
  return 0.03;
}
```

| 구역 레벨 | 초기 확률 | 라운드당 증가 | 최대 |
|---|---|---|---|
| Lv1-4 (쉬움) | 10% | +5%p | 50% |
| Lv5-7 (중간) | 7% | +4%p | 50% |
| Lv8-10 (어려움) | 5% | +3%p | 50% |

등장 시 → 다음 라운드 시작 시 초기 확률(baseRate)로 리셋.

### 6.3 쉬는 곳에서 가능한 것

**편성 변경 불가**(아군 교체·장비 교체·위치 변경 X). "그냥 대기실에서 쉬는" 분위기.

#### A. 자동 휴식 효과 (입장 시 즉시 발동)

| 효과 | 값 |
|---|---|
| 사망자 1명 부활 | HP 1%로 부활 (가장 최근에 죽은 1명) |
| 생존 아군 HP 회복 | 최대 HP의 +20% |

> 부활은 1명까지. 2명 이상 죽었으면 죽은 순서로 가장 최근 1명만.

#### B. 수동 아이템 사용 (선택)

수동 아이템(pocket) 슬롯에서 1회당 0~2개 사용 가능. 슬롯·아이템 종류는 §8 참고.

#### C. 진행

"다음 라운드로" 버튼 클릭 → 쉬는 곳 종료, R2 또는 R3 시작.

### 6.4 자동 모드(관전)에서

- 쉬는 곳 자동 효과는 동일 적용 (HP 회복 + 부활)
- 수동 아이템은 AI가 자동 사용 (HP 위급 시·다음 라운드 위협 시)
- 사용자 입력 없음

## 7. 길드회관 F (핏 통제) — v4 연동

자세한 단계 효과는 [guild-hall.md §10](guild-hall.md) 참고. v4에서 BP에 영향을 주는 핵심 단계:

| F 단계 | BP 영향 |
|---|---|
| F2 | 수동 아이템(pocket) 슬롯 해금 — 파티 공용 2칸 |
| F4 | **쉬는 곳 해금** (기본 확률표 적용) |
| F6 | **관중 난입 시스템 해금** (이전엔 핏 게이지가 33%/66% 넘어도 난입 없음 — 단순 보상 배율로만 작용) |
| F9 | 보스전(R3) 난입 확률 ×2 |
| F11+ | 쉬는 곳 등장 시 HP 회복 +20% (총 40%) |

> 즉 BP 첫 진입 시점에는 카드도 쉬는 곳도 난입도 없는 "순수 다키스트". F 트리를 키워가며 BP만의 색이 입혀짐. 이게 길드 성장 동기.

## 8. 수동 아이템 (pocket) 시스템

> v4 신규. 기존 `consumable`은 "전투 시작 시 자동 소비" 그대로 유지, 별개 시스템.

### 8.1 슬롯

- **파티 공용 2칸** (= 한 BP 런에서 파티 전체가 공유)
- 길드회관 F2 해금 필요
- 마을에서 사전에 채워서 출발

### 8.2 사용 시점

- **쉬는 곳에서만** 사용 가능
- 1회당 0~2개 사용 (소진 시 그 칸은 비움)

### 8.3 아이템 풀 (초안)

| 키 | 이름 | 효과 |
|---|---|---|
| `pot_heal_l` | 큰 치유 물약 | 아군 1명 HP 100% 회복 |
| `pot_heal_g` | 단체 치유 물약 | 아군 전체 HP 40% 회복 |
| `pot_revive` | 부활의 영약 | 사망자 1명 HP 50% 부활 (자동 부활 1%와 별개로 추가) |
| `pot_purify` | 정화의 향유 | 아군 전체 상태이상 제거 |
| `pot_atk` | 광기의 술 | 아군 전체 ATK +20% (다음 1R) |
| `pot_pit` | 피의 술잔 | 핏 게이지 즉시 +15% |

> 데이터는 추후 별도 파일(`data/pocketItems.js` 등)로 분리.

## 9. 서브 파견 vs 메인 전투 (v4)

| | 메인 전투 (수동) | 서브 파견 |
|---|---|---|
| 전투 방식 | 다키스트 4인 + 직접 스킬 | RunSimulator 시뮬 |
| 관중 난입 | 있음 (33%/66% 임계점) | 없음 |
| 쉬는 곳 | 있음 (확률 등장, F4+ 해금) | 없음 |
| 핏 게이지 | 있음 (보상 배율 + 난입 트리거) | 없음 (고정 배율) |
| 수동 아이템 | 사용 가능 (F2+) | 사용 불가 |
| 라운드 | 3라운드 | 시뮬 1회 |
