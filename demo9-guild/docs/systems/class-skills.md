# 시스템: 클래스 스킬

> 6클래스 각 1개 스킬. 메인 전투에서 **플레이어가 직접 발동**하는 핵심 능동 요소.
> 서브 파견에서는 사용 불가 (일반공격만) → 메인 vs 서브 차별화의 핵심.

## 1. 공통 룰

- **해금 레벨**: 용병 Lv 5
- **발동 방식**: 플레이어 탭/클릭 (수동) — 타이밍/대상 선택 = 실력
- **자동 모드**: 해당 구역+레벨 첫 클리어 후 AI 자동 발동 가능 (수동보다 비최적)
- **쿨다운**: 스킬마다 다름 (10~25초)
- **마나/코스트**: 없음. 쿨다운만 관리

## 2. 클래스별 스킬

### 🛡 전사 (warrior) — 방패막이

- **쿨다운**: 15초
- **효과**:
  - 대상: 자신 + 인접 아군 1명
  - 받는 피해 -40% (3초)
  - 적 도발 (3초간 자신을 우선 공격)
- **용도**: 위기 상황 방어, 사제/마법사 보호
- **레벨 보정**: Lv당 피해감소 +1% (Lv10 = -50%)

### 🗡 도적 (rogue) — 급소 찌르기

- **쿨다운**: 12초
- **효과**:
  - 대상: 단일 적
  - 즉시 일반 공격 1회 (확정 크리)
  - 출혈 (5초간 매초 ATK 20%)
- **용도**: 적 보스/엘리트 빠른 처치
- **레벨 보정**: Lv당 출혈 +2% (Lv10 = 매초 ATK 40%)

### 🔮 마법사 (mage) — 마력 폭발

- **쿨다운**: 20초
- **효과**:
  - 대상: 적 다수 (범위 120px)
  - 즉시 ATK × 2.5 피해
  - 화상 (3초간 매초 ATK 10%)
- **용도**: 적 무리 정리, 라운드 시작 폭딜
- **레벨 보정**: Lv당 피해 배율 +0.1 (Lv10 = ×3.5)

### 🏹 궁수 (archer) — 관통 사격

- **쿨다운**: 15초
- **효과**:
  - 대상: 일직선 관통
  - 경로상 모든 적에게 ATK × 2 피해
  - 첫 대상은 추가 ATK × 1 피해 (총 ×3)
- **용도**: 일렬 정렬된 적 일망타진, 후열 처치
- **레벨 보정**: Lv당 관통 거리 +20px

### ✝ 사제 (priest) — 신성 치유

- **쿨다운**: 18초
- **효과**:
  - 대상: 아군 전체
  - HP 25% 즉시 회복
  - 디버프 1개 해제
  - 3초간 받는 피해 -15%
- **용도**: 위기 회복, 디버프 해제
- **레벨 보정**: Lv당 회복량 +1% (Lv10 = 35%)

### ⚗ 연금술사 (alchemist) — 강화 물약

- **쿨다운**: 20초
- **효과**:
  - 대상: 아군 전체
  - ATK +25% (6초)
  - 공속 +15% (6초)
- **용도**: 보스전 폭딜 타이밍, 라운드 마무리
- **레벨 보정**: Lv당 버프 지속 +0.5초 (Lv10 = 11초)

## 3. 스킬 발동 UI

### 메인 전투 화면

```
하단 스킬 슬롯 (파티 인원수만큼):
  [🛡 ▣] [🗡 ▣] [🔮 ▣] [🏹 ▣] [✝ ▣] [⚗ ▣]
  쿨다운 회전 게이지 표시
  탭/클릭 = 즉시 발동
```

- 단축키 1~7로도 발동 가능
- 쿨다운 중에는 회색 처리
- 마우스 호버 시 효과 설명 툴팁

## 4. 스킬과 다른 시스템 연동

### 친화도 트리 노드
- BP 6A `도살자`: 스킬 피해 보스 대상 +15%
- 공통 `고급 전술` (Lv8A): 모든 스킬 쿨다운 -10%

### Bond 페어 스킬
- 티어 4+ 도달 쌍은 별도 페어 스킬 발동 (자동, 조건부)
- 페어 스킬은 클래스 스킬과 별개 슬롯 (UI 상단)

### 음성 특성
- `slow_thinker`: 스킬 쿨다운 +20%
- `quick_minded`: 스킬 쿨다운 -10%
- `nervous`: 위기 시 스킬 자동 발동 (수동 의도와 충돌 가능)

## 5. 자동전투 AI

자동 모드 (잠금해제 후) 발동 룰:
- 쿨다운 회복 즉시 자동 발동
- 우선순위는 클래스별로 다름:
  - 사제: 아군 HP 60% 이하 즉시
  - 전사: 자신 또는 아군 HP 40% 이하 즉시
  - 도적/마법사/궁수: 보스 또는 엘리트 등장 시 우선
  - 연금술사: 라운드 시작 5초 후
- 길드회관 D(자동화) 업그레이드로 AI 정확도 향상

## 6. 서브 파견에서

서브 파견은 일반공격만 사용. 스킬은 발동 안 됨.
- 이유: "스킬 없이 클리어할 수 없는 난이도"가 메인 전투의 의미
- RunSimulator는 스킬 효과를 계산에 포함 안 함 (파워 산정 시 일반공격 기준)
