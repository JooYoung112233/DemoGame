# 시스템: 피로도 (Stamina)

> 같은 용병만 굴리지 못하게 만들어 **로스터 회전을 강제**하는 핵심 트레이드오프.

## 1. 핵심 룰

- 모든 용병: stamina 0-100
- 시작 100, 파견/전투 후 감소
- 마을 시설에서 회복
- 같은 용병만 굴리면 자연히 피로도 누적 → 다른 용병 투입 강제

## 2. 피로도 차감

| 활동 | 차감 |
|---|---|
| 메인 직접 전투 (Blood Pit) | -25 |
| 메인 직접 전투 (Cargo) | -30 |
| 메인 직접 전투 (Blackout) | -40 |
| 서브 파견 (1런) | -12 ~ -20 (구역별) |
| 서브 파견 사망 위기 | -25 (생존 시 추가) |
| 메인에서 부상/HP낮음 종료 | +10 추가 차감 |

## 3. 피로도 효과

| Stamina | 효과 |
|---|---|
| 80-100 | 컨디션 최상 (+5% ATK) |
| 50-79 | 정상 |
| 30-49 | 피곤 (-10% ATK, -10% 명중) |
| 10-29 | 탈진 (-25% ATK, 파견 시간 +25%, 사망률 +50%) |
| 0-9 | 강제 휴식 (파견 불가) |

## 4. 피로도 회복

| 방법 | 회복량 | 비용 |
|---|---|---|
| 마을 자동 (기본) | +1/분 | 무료 |
| 마을 자동 (길드회관 C 풀) | +5/분 | 투자 필요 |
| 술집 휴식 | +30 즉시 | 20G |
| 신전 명상 | +50 즉시 | 50G |
| 술집 풀 휴식 | +100 즉시 | 100G |

## 5. gameState

```javascript
stamina: {}  // { mercId: 0~100 }
```
