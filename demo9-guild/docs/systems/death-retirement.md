# 시스템: 사망 / 은퇴

> 용병이 게임에서 영구히 떠나는 두 가지 경로. 다키스트던전식 영구사망과 자연스러운 은퇴.

## 1. 두 가지 경로

| 경로 | 트리거 | 영향 |
|---|---|---|
| 💀 **영구 사망** | 전투/파견 중 사망 | 영구 손실, Bond 보존, 추모비 |
| 🌅 **은퇴** | 자발적 또는 조건부 | 양도 보상, 전설 칭호 |

기본 모드: **사망 보호** (HP 0 → 부상 회복). 옵션으로 **영구사망 모드** 선택 가능.

## 2. 사망 시스템

### 2.1 사망 트리거

- 메인 전투: HP 0 + 부활 카드/패시브 없음
- 서브 파견: RunSimulator의 deathChance 발동 (시뮬 실패)
- 마을 이벤트: 일부 이벤트 결과 (영주 처형 등 극단적 경우)

### 2.2 기본 모드 — 사망 보호 (디폴트)

```
HP 0 → 부상 상태 진입
- 즉시 사망 회피
- 회복: 신전 부활 (비용: 길드Lv × 1000G)
- 회복 전까지 파견/전투 불가
- 길드회관 C(휴식) 업그레이드로 비용 ↓
```

부상 상태 효과:
- 모든 활동 불가
- 보관함에 표시 (회색 처리)
- 부활 후 5분간 피로도 페널티 (-30 시작)

### 2.3 영구사망 모드 (옵션)

- 게임 시작 또는 뉴게임+ 진입 시 선택
- HP 0 = **영구 사망** (부활 불가, 다키스트던전 스타일)
- 명예 포인트 보너스 +50% (도전 보상)
- 칭호: "철혈의 길드장"

### 2.4 사망 후 처리

| 항목 | 처리 |
|---|---|
| Bond | **영구 보존** (다른 용병에게 양도 불가) |
| 장비 | 자동 보관함 회수 |
| 친화도 | 소실 |
| 특성 | 소실 |
| 명예 포인트 | +10 (사망당, NG+ 진입 시 정산) |

### 2.5 추모비 (마을)

마을에 추모비 시설. 사망한 용병들의 이름/사망일/사망 원인 기록.

- 시설: 마을 광장 (자동 해금)
- 사망 용병 보유 데이터:
  - 이름, 클래스, 희귀도
  - 사망일 (게임 내 시간)
  - 사망 원인 (구역/보스/이벤트)
  - 최종 레벨
  - 보유 Bond 관계
- 효과: 추모비 방문 시 살아있는 용병 사기 +1 (소소한 버프)

## 3. 은퇴 시스템

### 3.1 은퇴 조건

자발적 / 조건부 / 강제 은퇴 3가지.

### 3.2 자발적 은퇴

플레이어가 결정. 어떤 용병이든 가능.

```
[은퇴 처리]
  대상 용병에게 "은퇴" 클릭
  → 확인 모달 (영구, 복구 불가)
  → 양도 보상:
    - 골드: Lv × 100
    - 양성 특성 1개 → 무작위 신규 용병에게 이월 (선택 가능)
    - 명예 포인트 +5
```

### 3.3 조건부 은퇴 (자동 트리거)

특정 조건 도달 시 은퇴 제안 이벤트 발생.

| 조건 | 효과 |
|---|---|
| Lv 30 도달 | "충분한 보상" — 은퇴 선택지 |
| 사이클 시간 절반 경과 + Lv 20 | "오랜 동반자" — 은퇴 권유 |
| 음성 특성 3개 누적 | "지친 영혼" — 명예 은퇴 (양도 보상 ×2) |
| Bond Max 쌍 보유 + 한쪽 사망 | "운명의 끝" — 은퇴 권유 |

### 3.4 강제 은퇴 (예외)

- 마을 이벤트 결과로 강제 은퇴 (예: 가족 위기 거부 시)
- NG+ 진입 시 모든 용병 자동 은퇴 (사이클 종료)

### 3.5 은퇴 보상

| 보상 | 효과 |
|---|---|
| 골드 | Lv × 100~300 (희귀도 배율) |
| 양성 특성 이월 | 최대 1개 → 새 용병 자동 부여 (선택) |
| 영구 칭호 | 길드 명예의 전당 (UI 기록) |
| 명예 포인트 | +5~20 (Lv/특성/Bond 비례) |
| 추모비 등록 | 명예 은퇴는 추모비에 같이 기록 |

## 4. 다른 시스템 연동

### Bond
- 사망 트리거 효과: 동반자 사망 시 살아있는 다른 한 명에게 ATK +20% (1분)
- 티어 5 Bond: 사망 시 동반자 1회 부활 (HP 30%)
- 사망 후에도 Bond 값 보존 (다른 용병에게 양도 불가)

### 길드 평판
- 사망 1명 당: -1 (전체 평판)
- 영구사망 모드: -3 (가중)
- 신전 부활 시: 사망 처리 안 됨 (부상만)

### 음성 특성
- `coward` (겁쟁이): 동반자 사망 시 5초 패닉
- `lone_wolf` (고독한 늑대): 사망 트리거 효과 안 받음

### 친화도 트리
- BO `dark_resilience` (Lv8): BO 사망률 -25%
- 친화도 트리는 용병 개인 → 사망 시 소실

## 7. 디자인 의도

- **선택 가능한 영구사망**: 다키스트던전식을 원하는 사람에게만
- **기본 안전**: 부상 회복이 기본 → 진입 장벽 낮음
- **은퇴 보상**: 키운 용병이 헛되지 않게 양도 보상
- **추모비**: 정서적 무게 (게임에 캐릭터가 살아있다는 느낌)
- **Bond 보존**: 사망해도 관계는 남는다 → 서사적 깊이
