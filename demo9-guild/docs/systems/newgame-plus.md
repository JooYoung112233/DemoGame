# 시스템: 영구 진행 / 뉴게임+ (Roguelike Meta Progression)

> 12시간 한 사이클 완주 후 뉴게임+ 진입. 메타 진행 + 도전 난이도 + 신규 콘텐츠 해금.

## 1. 사이클 완주 조건

엔드게임 트리거:
- 모든 3구역 보스 처치 완료
- 길드 평판 100 (왕실 인정) 도달
- 또는 단순히 12시간 누적 플레이 후 "사이클 종료" 선택

→ "왕실 접견" 시나리오 이벤트 → 뉴게임+ 진입 선택

## 2. 사이클 이월 vs 리셋

### 2.1 이월 (영구 보존)

뉴게임+ 진입 시 다음 항목은 유지:

| 항목 | 이월 |
|---|---|
| **명예 포인트** (HP) | ✅ 누적 |
| **영구 강화** | ✅ 모두 보존 |
| **해금 콘텐츠** | ✅ 해금 상태 유지 |
| **챌린지 진행** | ✅ |
| **업적** | ✅ |
| **칭호** | ✅ |
| **사이클 카운트** | ✅ +1 |

### 2.2 리셋

| 항목 | 리셋 |
|---|---|
| **골드** | ❌ → 시작 골드로 |
| **용병** | ❌ → 신규 모집 풀 |
| **장비/소재** | ❌ → 인벤토리 초기화 |
| **친화도 트리** | ❌ (단, 영구 강화로 시작 보너스 있음) |
| **길드 레벨/시설** | ❌ → 처음부터 |
| **NPC 호감도** | ❌ (단, 명예 포인트로 시작 호감 부여 가능) |
| **활성 의뢰** | ❌ |

## 3. 명예 포인트 (Honor Points, HP)

### 3.1 획득

사이클 완주 시 일괄 정산:

| 성과 | 명예 포인트 |
|---|---|
| 사이클 완주 기본 | +100 |
| 12시간 안에 완주 | +50 |
| 8시간 안에 완주 (스피드런) | +200 |
| 모든 구역 Lv10 보스 처치 | +150 |
| 평판 100 달성 | +50 |
| 모든 NPC 호감 60+ | +100 |
| 사이클 중 사망 없음 | +100 |
| 전설 장비 5개+ 보유 | +50 |
| 챌린지 클리어 (도전 모드) | +50~500 (도전별) |

### 3.2 사용

명예 포인트는 **영구 강화 상점**에서 사용:

| 영구 강화 | 비용 | 효과 |
|---|---|---|
| 시작 골드 +500 | 200 HP | 다음 사이클 시작 시 골드 +500 (스택 가능) |
| 시작 로스터 +1 | 500 HP | 시작 시 무료 용병 1명 (스택 ×3) |
| 시작 평판 +5 | 300 HP | 다음 사이클 시작 평판 (스택 ×4) |
| 시작 NPC 호감 +5 | 250 HP / NPC | 특정 NPC 시작 호감 (NPC별) |
| 시작 친화도 XP +500 | 400 HP / 구역 | 시작 시 친화도 일부 |
| 모든 스탯 +2% | 1000 HP | 영구 (스택 ×5 = 최대 +10%) |
| 시설 시작 해금 (1개) | 500 HP / 시설 | 시작부터 해당 시설 해금 |
| 사이클 시간 +1시간 | 800 HP | 12시간 → 13시간 (스택 ×3) |

## 4. 도전 난이도 (사이클 +1, +2, ...)

뉴게임+ 진입 시 도전 모드 선택 가능.

### 4.1 도전 모드 옵션

| 모드 | 효과 | 보상 |
|---|---|---|
| **일반** | 기본 난이도 | 기본 명예 포인트 |
| **도전 +1** | 적 +20% 강화 | 명예 ×1.3 |
| **도전 +2** | 적 +40% 강화, 사망률 ×1.5 | 명예 ×1.6 |
| **도전 +3** | 적 +60%, 사망률 ×2, 시설 비용 ×1.5 | 명예 ×2.0 |
| **악몽** | 적 ×2, 사망 영구, 자원 ×0.7 | 명예 ×3.0, 전설 칭호 |
| **커스텀** | 옵션 조합 (난이도 점수로 결정) | 점수 비례 |

### 4.2 사이클 카운트 보너스

매 사이클마다:
- 적 자동 +10% 강화 (NG+1, NG+2, ...)
- 시작 골드/명예 페이스 +5%
- 사이클 카운트가 10 도달 = "마스터" 칭호 + 영구 ATK +10%

## 5. 사이클별 신규 콘텐츠

뉴게임+ 진입 시 신규 콘텐츠 해금:

| 사이클 | 신규 콘텐츠 |
|---|---|
| NG+1 | 어려운 보스 페이즈 4 추가 (3페이즈→4페이즈) |
| NG+2 | 4번째 구역 "심연의 사원" 해금 |
| NG+3 | 새 클래스 "혈투사" 추가 |
| NG+4 | 신규 NPC 2명 (왕실 측근, 도적단 두목) |
| NG+5+ | 무한 도전 모드 (난이도 무한 상승) |

## 6. 업적 / 챌린지 시스템

### 6.1 챌린지 (사이클 내 도전)

| 챌린지 | 보상 |
|---|---|
| 사망 없이 BP 클리어 | 명예 +30 |
| 8시간 안에 사이클 완주 | 명예 +200 + 칭호 "스피드러너" |
| 전 NPC 호감 100 | 명예 +300 + 칭호 "친교의 달인" |
| 모든 전설 장비 1번씩 보유 | 명예 +500 + 칭호 "수집가" |
| 단일 용병으로 보스 1체 처치 | 명예 +200 + 칭호 "외로운 영웅" |
| 저주 레벨 10 BO 보스 처치 | 명예 +300 + 칭호 "심연을 건넌 자" |

### 6.2 칭호

명예 포인트로 영구 효과는 영구 강화 / 칭호는 시각적 표시 + 작은 효과.

## 9. 게이트

- 뉴게임+ 진입: 사이클 1 완주 (1회 도전 가능)
- 도전 모드: 사이클 1 완주 후 NG+에서 선택
- 악몽 모드: NG+2 클리어 후
- 신규 클래스: NG+3 진입 시
- 무한 도전: NG+5 완주 후

## 10. 디자인 의도

- **반복성 보장**: 12시간 한 번에 끝이 아닌, 여러 사이클로 게임 수명 연장
- **메타 보상**: 매번 0에서 시작하지만 영구 강화로 진행감 유지
- **도전 옵션**: 자신 있는 플레이어는 도전 +N으로 자기 시험
- **수집 동기**: 챌린지/칭호로 컴플리션 추구자 만족
- **새 콘텐츠**: 사이클마다 신선함 (신규 구역/클래스/NPC)
