# Demo 6: Cargo

## 컨셉
기차 방어 로그라이크. 3개 지역(폐허/설원/도시)을 관통하는 화물열차를 적 침입으로부터 방어한다. 각 칸에는 고유 기능이 있어 어떤 칸을 우선 방어할지가 핵심 전략.

## 핵심 루프
**편성 → 루트 선택 → 전투(칸별 방어) → 수리/업그레이드 → 다음 노드**

## 적 시스템

| 타입 | HP | ATK | 특징 |
|------|----|----|------|
| 크롤러 | 90 | 14 | 빠른 접근 |
| 탱커 | 300 | 22 | 문 파괴 특화, 느림 |
| 점퍼 | 120 | 18 | 후방 칸 침입 |
| 폭발물 | 60 | 80 | 자폭, 극딜 |

### 라운드별 적 구성
- **1~3라운드**: 크롤러 70%, 탱커 20%, 폭발물 10%
- **4~6라운드**: 크롤러 40%, 탱커 25%, 점퍼 20%, 폭발물 15%
- **7라운드+**: 크롤러 30%, 탱커 20%, 점퍼 25%, 폭발물 25%

### 침입 메커니즘
- 칸별 침입 확률: 기본 50% + 라운드×3% + 중요도 보너스
- 발전칸 +15%, 탄약칸 +10%, 문 파손 시 +15%
- 침입 경로: 창문 돌파, 천장 침입, 문 파괴, 외부 매달림

## 루트 맵 (3막)

### 1막 - 폐허 (적 스케일 1.0x)
- 노드 분포: 전투 50%, 엘리트 5%, 역 15%, 이벤트 20%, 휴식 10%
- 지형: 폐허(기본), 터널(시야 50%, 기습 30%), 다리(넉백, 좁은 전장)

### 2막 - 설원 (적 스케일 1.4x)
- 노드 분포: 전투 40%, 엘리트 15%, 역 15%, 이벤트 15%, 휴식 15%
- 지형: 설원(SPD -20%), 산악(엘리트 확률 30%), 숲(매복 20%, 회복 10%)

### 3막 - 도시 폐허 (적 스케일 1.9x)
- 노드 분포: 전투 35%, 엘리트 25%, 역 15%, 이벤트 10%, 휴식 15%
- 지형: 도시(호드 보너스, 전리품 1.5x), 터널, 다리

## 이벤트 시스템 (8종)

1. **생존자 발견**: 골드+30 (이후 전투 적+1) / 무시
2. **보급 캐시**: 부품+3, 의료킷+2 / 골드+20
3. **폭풍 접근**: 전 칸 HP -10% / 돌파(랜덤 칸 -25% HP, 골드+25)
4. **밀수업자**: 골드 30 → 랜덤 업그레이드 / 거절
5. **레일 파손**: 부품 2개 소모 안전 통과 / 강행(칸 피해)
6. **버려진 열차**: 부품+2, 골드+10 / 통과
7. **우호 캠프**: 의료킷 1 → 부품 3 / 다음 노드 공개
8. **미스터리 화물**: 50% 골드+40 / 50% 랜덤 칸 -20 HP / 무시
