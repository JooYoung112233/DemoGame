## 데모별 상세

### Demo 1: 라인 자동전투 (`demo1-line/`)

**장르**: 실시간 자동전투  
**컨셉**: 아군과 적이 1D 라인(Y=440) 위에서 자동 전투. 유저는 전투 전 조합/배치만 결정.

**핵심 메커니즘**
- 쿨타임 순환: 캐릭터별 고유 공격 속도 (도적 1.2s, 탱커 2s, 사제 7s, 마법사 10s)
- 스킬 연계: 도발→집중공격, 출혈→추가데미지, 힐+실드 타이밍
- 1D 충돌 시스템: 근접/원거리 포지셔닝, 팀 간 밀어내기

**캐릭터**
- 아군: 탱커, 도적, 사제, 마법사
- 적군: 고블린, 오거, 아처

**검증 포인트**: 쿨타임 순환과 스킬 콤보가 "조합을 바꿔보고 싶다"는 욕구를 만드는가?

### Demo 2: 턴 압축 전투 (`demo2-atb/`)

**장르**: ATB 게이지 시스템  
**컨셉**: 행동 게이지(0→1000)가 차면 자동 행동. 속도 스탯에 비례해 충전.

**핵심 메커니즘**
- ATB 게이지: 속도가 빠른 캐릭터가 더 자주 행동
- 턴 순서 전략: 도적 연타 빌드, 스턴 체인, 독 누적, 힐 사이클
- 빠른 템포: 실시간처럼 보이지만 턴 기반

**검증 포인트**: 속도 스탯 조작으로 턴 순서를 통제하는 전략적 쾌감이 있는가?

---

### Demo 3: 웨이브 생존 (`demo3-wave/`) — 미완성

**장르**: 로그라이크 서바이벌  
**컨셉**: 연속 적 웨이브를 자동전투로 버티며, 웨이브 사이에 강화 선택.

**핵심 메커니즘**
- 웨이브 진행: 점진적 난이도 상승 (수량, 속도, 구성)
- 중간 업그레이드: 공격/방어/하이브리드 선택지
- 보스: 웨이브 10에서 보스 등장
- "한 판 더" 중독 구조

**현재 상태**: 데이터/UI 파일은 존재하나 씬 로직 미구현

**검증 포인트**: 빌드 성장 선택과 점진적 압박이 반복 플레이 욕구를 만드는가?

---

### Demo 4: 호텔 100층 (`demo4-hotel/`)

**장르**: 로그라이트 액션  
**컨셉**: 1인 플레이어가 호텔 100층을 올라가며 전투. 층마다 무기/스킬 업그레이드.

**핵심 메커니즘**
- 층 기반 진행: 100층 = 높은 루프 잠재력
- 무기 시스템: 근접 무기별 다른 공격 패턴
- 층별 업그레이드 선택: 무기, 스킬, 스탯 강화
- 보스 층: 특정 층마다 보스 등장
- 이벤트 씬: 랜덤 이벤트/선택

**데이터**: weapons.js, enemies.js, upgrades.js, floors.js

**검증 포인트**: 점진적 층 진행 + 랜덤 업그레이드가 "한 층만 더"를 유발하는가?

### Demo 5: 블러드 핏 (`demo5-bloodpit/`)

**장르**: 아레나 서바이벌  
**컨셉**: 킬할수록 위험도(1~10)가 올라가는 사냥터. 언제 탈출할지 스스로 결정.

**핵심 메커니즘**
- DangerSystem: 킬마다 위험도 증가
  - Lv 0~2: 기본 러너 (3+ 마리)
  - Lv 3~4: 러너 + 브루저
  - Lv 5~6: 러너 + 브루저 + 스피터 + 둥지
  - Lv 7~9: 엘리트 + 보스 등장
  - Lv 10: 보스 + 부하 확정
- 적 파워 스케일링: 1 + (위험도 × 0.15)
- 루트 드랍: 킬 시 아이템 획득 (보관함에 적립)
- 탈출 선택: 쌓은 루트를 가지고 나갈 타이밍 결정

**공유 인프라**: ItemRegistry, StashManager, LootGenerator

**검증 포인트**: 손실 공포 vs 탐욕이 만드는 리스크/리워드 긴장감이 있는가?

---

### Demo 6: 라스트 카고 (`demo6-cargo/`)

**장르**: 열차 방어  
**컨셉**: 열차 칸마다 다른 기능. 적이 특정 칸을 공격. 칸 파괴 = 해당 기능 상실.

**핵심 메커니즘**
- 열차 칸별 기능:
  - 화물칸: 루트 드랍 (파괴 시 획득 불가)
  - 의무칸: 매 라운드 15% 힐 (파괴 시 힐 불가)
  - 탄약칸: 원거리 공격 활성화 (파괴 시 원거리 50% 약화)
  - 발전칸: 전체 모듈 전력 공급 (파괴 시 전체 모듈 비활성)
- InvasionSystem: 적이 특정 칸 타겟
- 연쇄 손실: 하나 잃으면 전략 전체가 흔들림
- 수리 씬: 손상된 칸 복구

**검증 포인트**: "어떤 기능을 포기할 것인가"의 희생 선택이 감정적 무게를 만드는가?

---

### Demo 7: 블랙 아웃 (`demo7-blackout/`)

**장르**: 탈출형 파밍  
**컨셉**: 어둠 속 시설에서 방을 돌며 파밍. 오래 머물수록 텐션 상승, 탈출이 어려워짐.

**핵심 메커니즘**
- TensionSystem (0~5단계):
  - 4방 이동마다 텐션 +1
  - 적 스케일링: 1 + (텐션 × 0.12)
  - 텐션 4 이상: 출구 위치 셔플 (함정)
  - 단계: 平穏 → 不安 → 危険 → 極度危険 → 恐怖 (색상 피드백)
- MapSystem: 다중 방 구조, 방마다 루트
- WeightSystem: 소지 용량 제한
- 방별 적 구성 차이
- 탈출 씬: 출구 자체가 점점 멀어짐

**검증 포인트**: 점진적 텐션이 "이 방까지만 더 갈까?"라는 불안 루프를 만드는가?

## 파밍/전투 추가 요소 제안

### A. 기존 데모 개선안

#### Demo 1 (라인 자동전투) — 시너지 파밍 추가
- **장비 드랍 시스템**: 전투 승리 시 장비 드랍. 같은 장비 3개 합성 → 상위 등급
- **시너지 보너스**: 특정 조합(탱커+사제, 도적+마법사)에 세트 효과 부여
- **일일 챌린지 적 구성**: 매번 다른 적 조합으로 메타 고정 방지

#### Demo 2 (ATB) — 속도 빌드 파밍
- **룬 시스템**: 전투 후 속도/공격 룬 드랍. 게이지 충전 패턴 커스터마이징
- **연계 보너스**: 특정 턴 순서로 행동하면 연계 보너스 데미지
- **속도 변동 이벤트**: 전투 중 랜덤으로 "가속 필드", "감속 필드" 등장

#### Demo 4 (호텔 100층) — 영구 성장 루프
- **소재 파밍**: 층 클리어 시 소재 드랍 → 영구 강화에 사용
- **무기 해금 트리**: 특정 층/조건 달성 시 새 무기 해금
- **엘리트 층**: 10층마다 엘리트 적 + 보장된 레어 드랍

#### Demo 5 (블러드 핏) — 위험도별 루트 테이블
- **위험도 전용 드랍**: 위험도 7+ 에서만 나오는 유니크 아이템
- **스태쉬 업그레이드**: 수집한 재화로 보관함 크기, 드랍률 영구 강화
- **도박 시스템**: 위험도 10 클리어 시 보너스 전리품 vs 전멸 리스크

#### Demo 6 (카고) — 열차 업그레이드 파밍
- **칸 업그레이드**: 수집 자원으로 칸별 기능 강화 (힐량 증가, 탄약 무한 등)
- **신규 칸 해금**: 특수 칸 (방어탑, 감시탑, 수리공방) 연구/해금
- **보급품 파밍**: 스테이지 사이 보급 이벤트 — 어떤 칸을 우선 수리할지 선택

#### Demo 7 (블랙아웃) — 탐색 보상 깊이
- **지도 조각 시스템**: 탐색할수록 숨겨진 방 위치가 드러남
- **무게 vs 가치 트레이드오프**: 무거운 고가치 아이템 vs 가벼운 저가치 다수
- **안전 장치 파밍**: 손전등 배터리, 지도, 소음기 등 다음 런을 위한 도구 수집

### B. 새로운 데모 컨셉 제안

#### Demo 8 컨셉: **"용병 시장"** (Mercenary Market)
**장르**: 파밍 경매 + 자동전투  
**핵심 루프**: 전투 → 루트 획득 → 시장에서 용병 고용/장비 구매 → 더 강한 전투

**고유 메커니즘**
- **경매 시스템**: 전투에서 얻은 금화로 용병/장비를 입찰 경쟁 (AI 경쟁자)
- **용병 계약**: 고용한 용병은 N전 후 계약 만료 → 재고용 or 새 용병
- **시세 변동**: 특정 용병/장비의 가격이 수요에 따라 변동
- **투자 판단**: 비싼 S급 용병 1명 vs 저렴한 B급 3명

**검증 질문**: 자원 배분(투자)의 판단이 전투 결과에 체감되는가?  
**파밍 깊이**: 금화 파밍 → 시장 탐색 → 최적 구매 → 전투력 검증의 사이클

#### Demo 9 컨셉: **"유전자 합성소"** (Gene Lab)
**장르**: 수집 합성 + 전투 테스트  
**핵심 루프**: 전투 → 유전자 드랍 → 합성 → 새로운 능력의 캐릭터 생성 → 전투 테스트

**고유 메커니즘**
- **유전자 조합**: 드랍된 유전자(공격형/방어형/속도형/특수형) 2~3개 합성
- **돌연변이**: 합성 시 랜덤 확률로 예상 밖 능력 발현
- **유전자 도감**: 수집한 조합 기록 → 도감 완성 욕구
- **전투 시험장**: 새로 만든 캐릭터를 즉시 전투에 투입해 성능 확인

**검증 질문**: "이 조합이면 뭐가 나올까?"라는 호기심이 반복 파밍을 유도하는가?  
**파밍 깊이**: 유전자 수집 → 조합 실험 → 결과 확인 → 도감 완성의 수집 사이클

#### Demo 10 컨셉: **"깊은 광산"** (Deep Mine)  
**장르**: 자원 채굴 + 전투 방어  
**핵심 루프**: 광맥 선택 → 채굴(시간 경과) → 몬스터 방어 → 자원으로 장비 제작

**고유 메커니즘**
- **광맥 선택**: 깊이별로 다른 광물 (깊을수록 희귀하지만 위험)
- **채굴 타이머**: 채굴 중 적이 공격 → 채굴을 중단하고 방어할지, 버틸지 판단
- **제작 시스템**: 채굴한 광물로 무기/방어구 제작
- **갱도 붕괴**: 깊이 한계를 넘기면 붕괴 위험 → 블랙아웃과 유사한 긴장감

**검증 질문**: 채굴과 방어의 동시 압박이 긴장감을 만드는가?  
**파밍 깊이**: 채굴 자원 관리 → 제작 → 장비 강화 → 더 깊은 층 도전

### C. 공통 파밍 시스템 제안 (전체 데모에 적용 가능)

| 시스템 | 설명 | 파밍 유발 요소 |
|--------|------|----------------|
| **도감 수집** | 적/아이템/조합 도감 | "빈 칸 채우기" 수집 욕구 |
| **일일 도전** | 매일 다른 조건의 전투 | 하루에 한 번 보상으로 접속 유도 |
| **업적 시스템** | 특정 조건 달성 시 영구 보너스 | 장기 목표 제공 |
| **소재 합성** | 하위 소재 N개 → 상위 소재 1개 | 반복 파밍의 명확한 동기 |
| **랭킹 보드** | 최고 기록 (층수, 위험도, 생존 웨이브) | 자기 경쟁 + 최적화 욕구 |
| **시즌 리셋** | 주기적으로 리셋 + 누적 보상 | 매 시즌 새로운 파밍 동기 |

## 전투 깊이 추가 요소

### 즉시 적용 가능한 요소
1. **속성 상성**: 불/얼음/번개 삼각 상성 → 적 구성에 따른 아군 조합 변화 강제
2. **크리티컬 체인**: 크리티컬 히트 시 다음 공격 크리티컬 확률 증가 → 연쇄 폭발 쾌감
3. **진형 효과**: 전열/후열 배치에 따른 보너스 (전열: 방어+, 후열: 공격+)
4. **약점 시스템**: 적마다 특정 공격에 약점 → 올바른 공격 매칭 시 2배 데미지

### 중기 적용 요소
5. **전투 중 장비 드랍**: 전투 중 적 사망 시 즉시 장비 드랍 → 줍기 판단
6. **환경 효과**: 맵마다 다른 환경 (용암: DoT, 빙판: 속도 감소, 안개: 명중률 감소)
7. **전투 중 이벤트**: 랜덤으로 "증원", "함정", "보물상자" 등장
8. **궁극기 게이지**: 팀 공유 게이지 → 가득 차면 강력한 합체기 발동
